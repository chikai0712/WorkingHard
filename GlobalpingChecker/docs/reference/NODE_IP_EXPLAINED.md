# 節點 IP 詳細說明

## 🔍 你看到的三種節點 IP

從你的測試結果中，出現了三種不同的節點 IP：

```
🔌 節點IP: private
🔌 節點IP: 1.1.1.1
🔌 節點IP: 2001:4860:4860::8888
```

---

## 📊 各種節點 IP 的含義

### 1. `private` - 隱私保護

```
🔌 節點IP: private
```

**含義**：
- 節點所有者選擇**隱藏真實 IP**
- Globalping 允許節點設置隱私保護
- 這是正常且合法的

**為什麼隱藏？**
- 保護節點隱私
- 避免被攻擊
- 企業內部節點不想暴露 IP

**是否影響測試？**
- ❌ **不影響**
- 測試功能完全正常
- 只是不顯示節點的真實 IP

**範例**：
```
📍 Tencent Building, Kejizh (AS132203) [Jakarta]
   🔌 節點IP: private         | 🎯 目標IP: 104.21.38.29
```

**解讀**：
- 這是騰訊在雅加達的節點
- 節點真實 IP 被隱藏
- 但成功測試了目標域名
- 目標 IP 是 104.21.38.29（Cloudflare）

---

### 2. `1.1.1.1` - Cloudflare DNS

```
🔌 節點IP: 1.1.1.1
```

**含義**：
- 這**不是**節點的真實 IP
- 這是 **Cloudflare 的公共 DNS 服務器**
- 節點可能使用了 DNS 代理或 VPN

**1.1.1.1 是什麼？**
- Cloudflare 的公共 DNS 服務
- 全球最快的 DNS 之一
- 類似 Google 的 8.8.8.8

**為什麼顯示 DNS IP？**
可能原因：
1. 節點使用了 Cloudflare WARP（VPN）
2. 節點通過 Cloudflare 代理
3. 節點配置了特殊的網絡設置

**是否影響測試？**
- ⚠️ **可能影響**
- 測試結果可能反映 Cloudflare 的網絡狀況
- 而不是真實的印尼 ISP 狀況

**範例**：
```
📍 Media Sarana Data (AS55666) [Samarinda]
   🔌 節點IP: 1.1.1.1         | 🎯 目標IP: 104.21.38.29
```

**解讀**：
- 這是 Media Sarana Data 在 Samarinda 的節點
- 但顯示的是 Cloudflare DNS IP
- 可能使用了 Cloudflare WARP
- 測試結果仍然有效，但可能不代表真實 ISP 狀況

---

### 3. `2001:4860:4860::8888` - Google DNS (IPv6)

```
🔌 節點IP: 2001:4860:4860::8888
```

**含義**：
- 這是 **Google 的公共 DNS 服務器（IPv6 版本）**
- IPv4 版本是 `8.8.8.8`
- 節點可能使用了 Google DNS 或 VPN

**2001:4860:4860::8888 是什麼？**
- Google Public DNS 的 IPv6 地址
- 對應的 IPv4 是 8.8.8.8
- 全球使用最廣泛的 DNS 之一

**為什麼顯示 DNS IP？**
可能原因：
1. 節點使用了 Google VPN
2. 節點配置了 Google DNS
3. 節點通過 Google 代理

**IPv6 vs IPv4**：
- `2001:4860:4860::8888` - IPv6 格式
- `8.8.8.8` - IPv4 格式
- 功能相同，只是協議不同

**是否影響測試？**
- ⚠️ **可能影響**
- 測試結果可能反映 Google 的網絡狀況
- 而不是真實的印尼 ISP 狀況

**範例**：
```
📍 Media Sarana Data (AS55666) [Yogyakarta]
   🔌 節點IP: 2001:4860:4860::8888 | 🎯 目標IP: 172.67.179.112
```

**解讀**：
- 這是 Media Sarana Data 在 Yogyakarta 的節點
- 但顯示的是 Google DNS IPv6 地址
- 可能使用了 Google VPN 或代理
- 測試結果仍然有效，但可能不代表真實 ISP 狀況

## 🔍 如何識別真實節點 IP

### 真實 IP 的特徵

```
🔌 節點IP: 183.60.82.98    ✅ 真實 IP
🔌 節點IP: 202.169.224.46  ✅ 真實 IP
🔌 節點IP: 103.49.221.211  ✅ 真實 IP
```

**特徵**：
- 不是 1.1.1.1 或 8.8.8.8
- 不是 private
- 不是 IPv6 DNS 地址
- 通常是 ISP 的 IP 段

### DNS/代理 IP 的特徵

```
🔌 節點IP: 1.1.1.1                  ⚠️ Cloudflare DNS
🔌 節點IP: 8.8.8.8                  ⚠️ Google DNS
🔌 節點IP: 2001:4860:4860::8888     ⚠️ Google DNS IPv6
🔌 節點IP: private                  ⚠️ 隱藏 IP
```
