# 啟動 GlobalpingChecker V4.1

## 快速啟動

請在**新的終端窗口**中執行以下命令：

```bash
cd /Users/ckchiu/Desktop/Project/GlobalpingChecker/v4.1
source venv/bin/activate
python -m uvicorn app.main:app --host 127.0.0.1 --port 8765
```

## 訪問應用

啟動成功後，在瀏覽器中打開：

**http://127.0.0.1:8765**

## 啟動成功的標誌

你應該會看到類似以下的輸出：

```
✅ PostgreSQL 資料庫初始化完成
📋 域名初始化: 新增 0 個, 已存在 499 個
🚀 GlobalpingChecker V4.1 已啟動
INFO:     Uvicorn running on http://127.0.0.1:8765 (Press CTRL+C to quit)
INFO:     Started server process [xxxxx]
INFO:     Waiting for application startup.
INFO:     Application startup complete.
```

## 如果端口被佔用

如果看到 `Address already in use` 錯誤，請更換端口號：

```bash
python -m uvicorn app.main:app --host 127.0.0.1 --port 8766
```

然後訪問 http://127.0.0.1:8766

## 功能測試

啟動後可以測試以下功能：

1. ✅ 查看總覽頁面 - 顯示循環狀態和統計
2. ✅ 點擊「正常區」、「異常區」、「待分類」標籤查看域名列表
3. ✅ 點擊域名列表中的「詳情」按鈕查看域名歷史
4. ✅ 點擊批次列表中的「查看詳情」查看批次結果
5. ✅ 在「節點驗證」標籤中驗證 IP 地址

## 停止應用

在運行應用的終端中按 `Ctrl+C` 即可停止。
