## 📈 功能特性

### 智能循環檢測
- **第一循環** (AM 1:00 - AM 9:00): 檢測異常區 + 待分類區
- **第二循環** (AM 9:00 - AM 1:00): 檢測正常區
- **檢測間隔**: 90 分鐘
- **自動切換**: 根據時間自動切換循環

### 域名區域管理
- **正常區**: 連續檢測正常的域名
- **異常區**: 檢測異常的域名，需要持續監控
- **待分類**: 尚未檢測的新域名

### Web Dashboard
- 實時顯示循環狀態和統計數據
- 查看各區域域名列表
- 查看域名檢測歷史
- 查看批次結果詳情
- 節點 IP 驗證功能

## 🎉 部署優勢

### 相比舊版本的改進
1. **無需 PostgreSQL** - 使用 SQLite，部署更簡單
2. **智能循環** - 根據時間自動切換檢測策略
3. **區域管理** - 自動分類和管理域名
4. **更好的 UI** - 現代化的 Web Dashboard
5. **節點驗證** - 可以驗證檢測節點的地理位置
6. **詳細歷史** - 完整的檢測歷史和區域變更記錄

### 技術棧
- **後端**: FastAPI + SQLAlchemy
- **前端**: Alpine.js + Tailwind CSS
- **數據庫**: SQLite
- **調度器**: APScheduler
- **容器化**: Docker + Docker Compose
- **時區**: GMT+8 (Asia/Taipei)
