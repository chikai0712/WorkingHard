## 功能說明

### 主要功能
1. **總覽頁面** - 查看循環狀態和統計數據
2. **正常區** - 查看連續正常的域名列表
3. **異常區** - 查看需要監控的異常域名
4. **待分類** - 查看尚未檢測的域名
5. **節點驗證** - 驗證檢測節點的地理位置

### 詳細資訊功能
- 點擊域名列表中的「詳情」按鈕可查看：
  - 域名檢測歷史
  - 節點檢測詳情
  - 區域變更歷史
  
- 點擊批次列表中的「查看詳情」可查看：
  - 批次中所有域名的檢測結果
  - 每個域名的節點詳情
  - 可按狀態篩選結果

## 故障排除

### 如果端口被佔用
修改 `start.sh` 中的端口號（例如改為 9001）：
```bash
python -m uvicorn app.main:app --host 127.0.0.1 --port 9001 --reload
```

### 如果數據庫錯誤
1. 檢查數據庫文件權限：`ls -l data/globalping_results.db`
2. 運行測試腳本：`python test_db.py`
3. 如需重新遷移，運行：`python migrate_database.py`

## 技術棧

- **後端**: FastAPI + SQLAlchemy
- **前端**: Alpine.js + Tailwind CSS
- **數據庫**: SQLite
- **調度器**: APScheduler

## 注意事項

1. 應用會自動在 AM 1:00 和 AM 9:00 切換檢測循環
2. 每 90 分鐘自動執行一次檢測
3. 可以手動點擊「立即檢測」按鈕觸發檢測
4. 所有檢測結果都會保存到數據庫中
