## 📝 建議

### 定期清理
建議定期執行以下清理：

```bash
# 刪除 .DS_Store 文件
find . -name ".DS_Store" -delete

# 刪除 Python 緩存
find . -name "__pycache__" -type d -exec rm -rf {} +
find . -name "*.pyc" -delete

# 刪除舊備份（保留最近 3 個）
ls -t *.backup.* | tail -n +4 | xargs rm -f
```
