# ⚔️ 荒野亂鬥 - Brawl Stars

一個可以自訂角色的戰鬥遊戲，支援 AI 生成新角色。

## ✨ 功能特色

### 角色管理
- ➕ **新增角色** - 手動創建自訂角色
- ✏️ **編輯角色** - 修改現有角色屬性
- 🗑️ **刪除角色** - 移除不需要的角色
- 🔍 **搜尋角色** - 快速找到想要的角色

### AI 生成角色
- 🤖 **智能生成** - 根據描述自動生成角色
- 🎨 **風格選擇** - 選擇角色類型（戰士、法師、弓箭手等）
- ⚡ **自動屬性** - AI 自動分配合理的屬性值
- 💾 **一鍵保存** - 生成後直接保存到角色庫

### 遊戲系統
- ⚔️ **戰鬥系統** - 回合制戰鬥
- 🛡️ **防禦機制** - 使用防禦減少傷害
- ✨ **特殊技能** - 每個角色都有獨特技能
- 📊 **屬性系統** - 生命值、攻擊力、防禦力、速度

## 🚀 使用方法

### 本地運行

1. 打開 `index.html` 檔案即可使用

或使用本地伺服器：

```bash
cd BrawlStars
python3 -m http.server 8000
```

2. 在瀏覽器中打開 `http://localhost:8000`

### 基本操作

#### 選擇角色
1. 在主界面瀏覽可用角色
2. 點擊選擇一個角色
3. 點擊「🎮 開始遊戲」

#### 新增角色
1. 點擊「👤 角色管理」
2. 點擊「➕ 新增角色」
3. 填寫角色資訊
4. 點擊「儲存」

#### AI 生成角色
1. 點擊「🤖 AI 生成角色」
2. 輸入角色描述（例如：「一個擁有火焰能力的戰士，速度快，攻擊力高」）
3. 選擇角色風格
4. 點擊「🎨 生成角色」
5. 查看生成結果，點擊「✅ 儲存角色」

#### 遊戲操作
- **⚔️ 攻擊** - 對敵人造成傷害
- **🛡️ 防禦** - 減少受到的傷害
- **✨ 技能** - 使用特殊技能造成更多傷害
- **🏁 結束遊戲** - 返回角色選擇界面

## 🎮 角色屬性說明

- **生命值 (Health)** - 角色的血量，歸零則失敗
- **攻擊力 (Attack)** - 每次攻擊造成的傷害
- **防禦力 (Defense)** - 減少受到的傷害
- **速度 (Speed)** - 角色的移動和反應速度

## 🤖 AI 生成提示詞技巧

### 描述關鍵詞
- **元素**: 火、冰、雷、暗、光
- **屬性**: 快、慢、高攻擊、高防禦、高生命
- **風格**: 戰士、法師、弓箭手、坦克、刺客

### 範例提示詞
- "一個擁有火焰能力的戰士，速度快，攻擊力高"
- "防禦力極高的冰霜坦克，生命值高"
- "速度極快的暗影刺客，高攻擊力"
- "精通雷電魔法的法師，攻擊力高"

## 🎨 自訂角色

### 角色資料結構
```javascript
{
    id: "唯一ID",
    name: "角色名稱",
    description: "角色描述",
    health: 生命值,
    attack: 攻擊力,
    defense: 防禦力,
    speed: 速度,
    color: "顏色代碼",
    special: "特殊技能名稱",
    emoji: "表情符號",
    image: "圖片URL（可選）"
}
```

### 手動新增角色範例
1. 進入角色管理
2. 點擊「新增角色」
3. 填寫以下資訊：
   - 名稱：火焰劍士
   - 生命值：150
   - 攻擊力：35
   - 防禦力：12
   - 速度：65
   - 顏色：選擇紅色
   - 特殊技能：火焰斬擊

## 🔧 技術特點

- **純前端實現** - 無需後端服務器
- **本地儲存** - 使用 localStorage 保存角色資料
- **AI 生成** - 智能解析提示詞生成角色
- **Canvas 繪圖** - 使用 Canvas API 繪製遊戲畫面
- **模組化設計** - 代碼結構清晰，易於擴展
