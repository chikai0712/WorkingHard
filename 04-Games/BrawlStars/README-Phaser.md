# ⚡ Phaser 3 版本 - 快速開始指南

## 🎮 核心機制

這個版本完全實現了《荒野亂鬥》的核心機制：

### ✅ 已實現的功能

1. **WASD 移動**
   - W/S/A/D 或方向鍵控制
   - 平滑的移動體驗
   - 世界邊界碰撞

2. **滑鼠指向射擊**
   - 點擊滑鼠發射子彈
   - 子彈朝向滑鼠位置
   - 即時射擊響應

3. **三格彈藥系統** ⭐
   - 最多 3 發彈藥
   - 每次射擊消耗 1 發
   - 彈藥用盡無法射擊

4. **自動回復彈藥** ⭐
   - 每 1.5 秒回復 1 發彈藥
   - 視覺化的彈藥條顯示
   - 裝彈動畫效果

## 🚀 快速開始

### 方法 1：直接打開
```bash
# 直接雙擊打開
game-phaser.html
```

### 方法 2：從主界面進入
1. 打開 `index.html`
2. 點擊「⚡ Phaser 版本」按鈕

### 方法 3：本地伺服器
```bash
cd BrawlStars
python3 -m http.server 8000
# 訪問 http://localhost:8000/game-phaser.html
```

## 🎯 操作說明

- **移動**：W/A/S/D 或方向鍵
- **射擊**：滑鼠點擊
- **彈藥**：自動回復（1.5秒/發）

## 💡 彈藥系統說明

### 彈藥限制
- 開始時有 3 發彈藥
- 每次射擊消耗 1 發
- 彈藥用盡時無法射擊

### 自動回復
- 每 1.5 秒回復 1 發彈藥
- 如果持續射擊，會等待彈藥回復
- 射擊時會重置裝彈計時器

### 視覺反饋
- 彈藥條顯示當前彈藥數量
- 正在裝彈的格子會有脈衝動畫
- 實時更新裝彈進度

## 📝 程式碼說明

### 彈藥系統實現
```javascript
let ammo = 3;              // 當前彈藥
let maxAmmo = 3;           // 最大彈藥
let reloadTimer = 0;       // 裝彈計時器
let reloadTime = 1500;     // 裝彈時間（1.5秒）

// 自動回復邏輯
if (ammo < maxAmmo) {
    reloadTimer += delta;
    if (reloadTimer >= reloadTime) {
        ammo++;
        reloadTimer = 0;
    }
}
```

### 射擊檢查
```javascript
function fireBullet(pointer, shooter, bulletGroup, bulletColor) {
    // 檢查彈藥（核心機制）
    if (shooter === player && ammo <= 0) {
        return; // 沒有彈藥，無法射擊
    }
    
    // ... 發射子彈
    
    // 消耗彈藥
    if (shooter === player) {
        ammo--;
        reloadTimer = 0; // 重置計時器
    }
}
```

## 🎨 UI 元素

### 彈藥條
- 三個格子代表三發彈藥
- 實心圓點表示有彈藥
- 空格子表示無彈藥
- 裝彈時有脈衝動畫

### 生命值顯示
- 顯示當前生命值 / 最大生命值
- 受到傷害時角色會變紅
- 生命值歸零時遊戲結束

## 🔮 可以擴展的功能

基於這個原型，可以輕鬆添加：

1. **牆壁和障礙物**
   ```javascript
   // 添加靜態物理體
   walls = this.physics.add.staticGroup();
   // 子彈碰到牆壁會消失
   ```

2. **草叢隱身機制**
   ```javascript
   // 檢測是否在草叢區域
   if (inBush) {
       player.setAlpha(0.5); // 半透明
   }
   ```

3. **更多敵人類型**
   ```javascript
   // 不同的 AI 行為模式
   enemy.aiType = 'aggressive' | 'defensive' | 'patrol';
   ```

4. **特殊技能**
   ```javascript
   // 特殊技能系統
   specialAbility() {
       // 技能邏輯
   }
   ```
