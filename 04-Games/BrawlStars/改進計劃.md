## 📝 詳細改進項目

### 1. 即時戰鬥系統

#### 當前問題
- 回合制戰鬥缺乏即時性
- 無法體現操作技巧
- 戰鬥節奏較慢

#### 改進方案
```javascript
// 新增移動控制
class MovementController {
    handleKeyPress(key) {
        // W/A/S/D 或方向鍵控制移動
    }
    
    handleTouchMove(x, y) {
        // 觸控移動
    }
}

// 即時攻擊系統
class AttackSystem {
    attack(attacker, direction) {
        // 即時計算傷害和命中
        // 添加攻擊動畫
    }
}
```

### 2. 射程系統實現

#### 屬性定義
```javascript
{
    range: 5, // 1-10，近戰到遠程
    attackType: 'ranged', // 'melee' | 'ranged' | 'area'
    projectileSpeed: 10, // 子彈速度（僅遠程）
    attackAngle: 60 // 攻擊角度（範圍攻擊）
}
```

#### 攻擊判斷
```javascript
function canAttack(attacker, target) {
    const distance = calculateDistance(attacker, target);
    return distance <= attacker.range;
}
```

### 3. 角色類型擴展

#### 新增角色類型
- **治療者 (Healer)**: 可以治療隊友
- **控制者 (Controller)**: 具有控制技能
- **召喚師 (Summoner)**: 可以召喚寵物

### 4. 地圖系統設計

#### 地圖元素
- **牆壁**: 阻擋攻擊和移動
- **草叢**: 隱藏角色視野
- **補給點**: 恢復生命值
- **陷阱**: 造成傷害或減速

#### 地圖類型
- **開闊地圖**: 適合遠程角色
- **複雜地圖**: 適合近戰和刺客
- **對稱地圖**: 公平競技

## 💻 技術實現建議

### Canvas 優化
- 使用 requestAnimationFrame 實現流暢動畫
- 實現對象池減少內存分配
- 使用離屏 Canvas 優化渲染

### 碰撞檢測
- 使用圓形碰撞檢測（簡單高效）
- 實現空間分割優化性能

### 遊戲循環
```javascript
function gameLoop() {
    update(deltaTime);
    render();
    requestAnimationFrame(gameLoop);
}
```
