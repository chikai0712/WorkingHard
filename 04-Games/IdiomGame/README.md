# 📚 成語遊戲

一個互動式的成語學習遊戲，包含多種遊戲模式，讓你在玩樂中學習成語。

## 🎮 遊戲模式

### 1. 填空題
根據提示填寫成語中的空格，例如「一___二___」

### 2. 猜意思
根據成語選擇正確的含義

### 3. 接龍
根據前一個成語的最後一個字，接出下一個成語

### 4. 找成語
從一段文字中找出隱藏的成語

## 🚀 使用方法

### 方法 1：直接打開
```bash
# 雙擊打開
index.html
```

### 方法 2：本地伺服器
```bash
cd IdiomGame
python3 -m http.server 8000
# 訪問 http://localhost:8000
```

## 🎯 計分規則

- **答對**：+10 分
- **答錯**：-5 分
- **跳過**：-3 分
