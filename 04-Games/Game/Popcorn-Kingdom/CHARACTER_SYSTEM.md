# 🎭 角色運送系統說明

## 🌟 新功能：三麗鷗角色運送

遊戲現在加入了**角色運送系統**！可愛的三麗鷗角色會在畫面上走動，運送物品到各個建築。

---

## 👥 四位三麗鷗角色

### 🐶 大耳狗 (Cinnamoroll)
- **位置**：玉米田旁邊
- **職責**：守護玉米田
- **特色**：可愛的藍白色飛行狗狗

### 😈 庫洛米 (Kuromi)
- **位置**：市場旁邊
- **職責**：運送玉米到市場販售
- **工作流程**：
  1. 走到玉米田拾取玉米 🌽
  2. 運送到市場
  3. 賣出玉米賺錢 💰
  4. 返回市場待命

### 🐶 布丁狗 (Pompompurin)
- **位置**：烤爐旁邊
- **職責**：運送玉米到烤爐烤製
- **工作流程**：
  1. 走到玉米田拾取玉米 🌽
  2. 運送到烤爐
  3. 烤成爆米花 🍿
  4. 返回烤爐待命

### 🐰 美樂蒂 (My Melody)
- **位置**：外送站旁邊
- **職責**：運送爆米花到外送站配送
- **工作流程**：
  1. 走到烤爐拾取爆米花 🍿
  2. 運送到外送站
  3. 外送賺錢 💰
  4. 返回外送站待命

## 🎮 運送機制

### 自動運送
- 當建築準備好處理物品時，對應的角色會**自動**開始運送
- 你不需要點擊任何東西，只需要觀察角色走動

### 運送動畫
- 角色走動時會有**行走動畫**（上下跳動）
- 拾取物品後，物品會顯示在角色頭上
- 角色會自動轉向（面向移動方向）

### 運送時間
- 運送時間取決於建築之間的距離
- 角色移動速度：
  - 大耳狗：150 像素/秒
  - 庫洛米：120 像素/秒
  - 布丁狗：130 像素/秒
  - 美樂蒂：140 像素/秒

## 🎯 策略建議

### 1. 觀察角色狀態
- **待機**：角色站在建築旁邊
- **行走**：角色正在運送物品
- **攜帶物品**：角色頭上有物品圖示

### 2. 升級建議
- 升級建築可以**加快生產速度**
- 但角色運送時間不變
- 建議平衡升級，避免某個環節太慢

### 3. 多條路線
- 庫洛米和布丁狗可以**同時**從玉米田取玉米
- 如果玉米不夠，角色會排隊等待
- 建議升級玉米田提高產量

### 4. 瓶頸分析
- 如果角色經常待機 = 資源不足
- 如果資源滿了 = 處理速度太慢
- 觀察角色忙碌程度來決定升級方向

## 💡 有趣的細節

### 角色個性
- 每個角色有不同的移動速度
- 角色會自動轉向面對移動方向
- 攜帶物品時會有彈跳動畫

### 視覺回饋
- 角色名字顯示在下方
- 攜帶的物品顯示在角色旁邊
- 行走時有上下跳動動畫

### 任務隊列
- 如果角色正在忙碌，新任務會加入隊列
- 角色完成當前任務後會自動執行下一個
- 最多可以排隊等待

---

## 🎨 視覺效果

### 角色樣式
- 大圖示（40px）
- 名字標籤（白色背景）
- 物品圖示（24px，帶彈跳動畫）
- 陰影效果

### 動畫效果
- **行走動畫**：上下跳動（0.4秒循環）
- **物品彈跳**：持續彈跳（0.5秒循環）
- **轉向動畫**：水平翻轉（0.3秒過渡）

## 🔧 技術細節

### 角色系統架構
```javascript
Character 類別：
- 位置和移動
- 任務管理
- 物品攜帶
- DOM 元素控制

CharacterManager 類別：
- 管理所有角色
- 創建運送任務
- 任務隊列
- 更新角色狀態
```

### 運送流程
```javascript
1. 建築觸發 needDelivery 事件
2. 遊戲創建運送任務
3. 角色移動到取貨點
4. 拾取物品
5. 移動到目的地
6. 放下物品
7. 執行建築邏輯
8. 返回待機位置
```
