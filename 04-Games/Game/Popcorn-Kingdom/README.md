# 🏰 爆米花王國 - 遊戲設計文檔

## 🎯 遊戲概念
一個適合小學二年級的自動化經營遊戲。玩家在城堡中建立玉米產業鏈，從種植到販售，賺錢建設王國。

## 🎮 核心玩法
- **自動化生產**：建築自動運作，玩家專注於策略規劃
- **三條路線**：直接賣玉米 / 烤爆米花 / 外送服務
- **升級系統**：用賺來的錢升級建築，提升效率
- **建設系統**：蓋房子賺被動收入，種花提升效果

## 🏗️ 建築系統

### 1. 玉米田 🌾（免費，自動解鎖）
**功能**：自動生產玉米
- 等級 1：5秒/個，儲存10個（免費）
- 等級 2：3秒/個，儲存20個（50金幣）
- 等級 3：2秒/個，儲存30個（150金幣）
- 等級 4：1秒/個，儲存50個（500金幣）

**品質系統**：
- 🌽 好玉米：70%（可升級到95%）
- 🥀 壞玉米：30%（會被退貨）

### 2. 市場 🏪（免費，自動解鎖）
**功能**：自動販售玉米
- 等級 1：3秒賣1個（免費）
- 等級 2：2秒賣1個（100金幣）
- 等級 3：1秒賣1個（300金幣）
- 等級 4：同時賣2個（1000金幣）

**收益**：
- 好玉米：+10金幣
- 壞玉米：-5金幣，滿意度-10%

**客戶滿意度影響**：
- 😊 滿意(80-100%)：價格×1.0
- 😐 普通(50-79%)：價格×0.8
- 😠 生氣(0-49%)：價格×0.5

### 3. 烤爐 🔥（100金幣解鎖）
**功能**：將玉米烤成爆米花
- 等級 1：5秒烤1個（100金幣解鎖）
- 等級 2：3秒烤1個（200金幣）
- 等級 3：2秒烤1個（500金幣）
- 等級 4：同時烤2個（1500金幣）

**轉換**：
- 好玉米 → 完美爆米花
- 壞玉米 → 燒焦（浪費）

### 4. 外送站 🚚（200金幣解鎖）
**功能**：外送爆米花賺高收益
- 等級 1：3秒送1個（200金幣解鎖）
- 等級 2：2秒送1個（400金幣）
- 等級 3：1秒送1個（1000金幣）
- 等級 4：收益+50%（3000金幣）

**收益**：
- 完美爆米花：+50金幣

**訂單系統**：
- 每60秒出現特殊訂單
- 小訂單：3個→+30金幣
- 中訂單：5個→+60金幣
- 大訂單：10個→+150金幣

### 5. 建設系統

**房子（被動收入）**：
- 🏠 小屋：100金幣 → 每秒+1金幣
- 🏡 房子：500金幣 → 每秒+5金幣
- 🏰 豪宅：2000金幣 → 每秒+20金幣

**花園（提升效果）**：
- 🌸 小花：50金幣 → 滿意度+5%
- 🌺 大花：200金幣 → 滿意度+10%
- 🌳 大樹：1000金幣 → 玉米品質+10%
- ⛲ 噴泉：5000金幣 → 所有收益+20%

## 📊 遊戲數值平衡

### 收益對比
- 直接賣玉米：10金幣/個（快速但低收益）
- 烤爆米花：50金幣/個（5倍收益，需投資）
- 外送服務：50金幣/個 + 訂單獎勵（最高收益）

### 升級投資回報
- 玉米田升級：提升產量，快速回本
- 市場升級：提升銷售速度
- 烤爐升級：提升加工速度
- 外送站升級：提升收益

### 建設投資回報
- 房子：被動收入，長期投資
- 花園：提升效果，戰略投資

4. **耐心培養**：等待自動化系統運作
