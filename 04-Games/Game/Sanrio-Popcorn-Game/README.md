# 🎀 三麗鷗爆米花大遊行

一個可愛的多階段連鎖反應遊戲，玩家需要同時管理四個生產階段，確保爆米花生產線順暢運作！

## 🎮 遊戲特色

- **四個可愛的三麗鷗角色**：大耳狗、庫洛米、布丁狗、美樂蒂
- **多階段連鎖反應**：每個階段環環相扣
- **Fever Time 系統**：爽快的爆發時刻
- **連擊系統**：考驗反應速度和準確度
- **可愛的視覺風格**：三麗鷗經典粉嫩色調

## 🎯 遊戲玩法

### 階段 1：🐶 大耳狗 - 原料接取
- **操作**：滑鼠左右移動 / 鍵盤 ← → 鍵
- **目標**：接住掉落的玉米（避開壞玉米和石頭）
- **得分**：
  - 好玉米 🌽：+10 分
  - 金色玉米 ⭐：+30 分
  - 壞玉米 🥀：-5 分
  - 石頭 🪨：-10 分

### 階段 2：😈 庫洛米 - 魔法加熱
- **操作**：快速點擊滑鼠 / 空白鍵連打
- **目標**：將溫度維持在黃金區間（60-80°C）
- **得分**：完美爆發 +50 分

### 階段 3：🐶 布丁狗 - 精準包裝
- **操作**：看準時機按空白鍵 / 點擊滑鼠
- **目標**：在爆米花飛到目標區時點擊
- **得分**：
  - Perfect：+20 分
  - Great：+15 分
  - Good：+10 分
  - 連擊加成：每 5 連擊 +10%

### 階段 4：🐰 美樂蒂 - 裝箱送餐
- **操作**：拖曳爆米花到紙箱
- **目標**：裝滿 6 個爆米花並送給客人
- **得分**：成功服務客人 +100 分

## ⌨️ 操作說明

### 鍵盤快捷鍵
- **1/2/3/4**：快速切換到對應階段
- **← →**：移動大耳狗（階段 1）
- **空白鍵**：加熱（階段 2）/ 包裝（階段 3）
- **ESC**：暫停遊戲

### 滑鼠操作
- **移動**：控制大耳狗移動（階段 1）
- **點擊**：加熱（階段 2）/ 包裝（階段 3）
- **拖曳**：裝箱（階段 4）

## 🌟 Fever Time

### 觸發條件
- Fever 計量表達到 100%

### 效果（持續 15 秒）
- 🐶 大耳狗：自動吸引玉米
- 😈 庫洛米：自動維持完美溫度
- 🐶 布丁狗：所有判定都是 Perfect
- 🐰 美樂蒂：客人耐心凍結
- 💰 所有分數 x3
