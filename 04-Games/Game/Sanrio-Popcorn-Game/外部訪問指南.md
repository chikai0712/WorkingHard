# 🌐 三麗鷗爆米花大遊行 - 外部訪問指南

## 🚀 快速啟動（外部訪問）

### 步驟 1：啟動伺服器

在終端執行以下命令：

```bash
cd /Users/ckchiu/Desktop/Project/Game/Sanrio-Popcorn-Game
python3 -m http.server 8888 --bind 0.0.0.0
```

**重要參數說明**：
- `8888` - 端口號（可以改成其他，如 3000, 8080 等）
- `--bind 0.0.0.0` - 允許所有網路介面訪問（包括外部）

---

### 步驟 2：獲取您的 IP 位址

#### 方法 A：使用終端
```bash
ifconfig | grep "inet " | grep -v 127.0.0.1
```

#### 方法 B：使用系統偏好設定
1. 打開「系統偏好設定」
2. 點擊「網路」
3. 選擇您正在使用的連線（Wi-Fi 或乙太網路）
4. 查看「IP 位址」

#### 方法 C：快速查看
```bash
ipconfig getifaddr en0    # Wi-Fi
# 或
ipconfig getifaddr en1    # 乙太網路
```

---

### 步驟 3：訪問遊戲

#### 本機訪問（您自己的電腦）
```
http://localhost:8888
```

#### 區域網路訪問（同一個 Wi-Fi 的其他設備）
```
http://您的IP位址:8888
```

例如：
- `http://192.168.1.100:8888`
- `http://10.0.0.50:8888`

#### 外網訪問（需要額外設定）
```
http://您的公網IP:8888
```

## 🌍 外網訪問設定（選擇性）

如果您想讓外網（不同網路）的人也能訪問，需要：

### 1. 路由器端口轉發設定

登入您的路由器管理介面（通常是 `192.168.1.1` 或 `192.168.0.1`）：

1. 找到「端口轉發」或「Port Forwarding」設定
2. 新增規則：
   - **外部端口**：8888
   - **內部 IP**：您的電腦 IP（如 192.168.1.100）
   - **內部端口**：8888
   - **協議**：TCP
3. 儲存設定

### 2. 獲取公網 IP

訪問以下網站查看您的公網 IP：
- https://www.whatismyip.com/
- https://ipinfo.io/ip

或使用命令：
```bash
curl ifconfig.me
```

### 3. 分享連結

將以下連結分享給外網使用者：
```
http://您的公網IP:8888
```

## 🐛 常見問題

### Q1: 無法訪問？
**檢查清單**：
- [ ] 伺服器是否正在運行？
- [ ] IP 位址是否正確？
- [ ] 端口號是否正確？
- [ ] 防火牆是否允許？
- [ ] 是否在同一個網路？

### Q2: 外網無法訪問？
**檢查清單**：
- [ ] 路由器端口轉發是否設定？
- [ ] 公網 IP 是否正確？
- [ ] ISP 是否封鎖端口？
- [ ] 是否有動態 IP（需要 DDNS）？

### Q3: 連線很慢？
**可能原因**：
- 網路頻寬不足
- 太多人同時訪問
- 電腦效能不足

**解決方案**：
- 使用有線網路
- 限制同時訪問人數
- 升級網路方案

## 💡 進階選項

### 使用不同端口
```bash
python3 -m http.server 3000 --bind 0.0.0.0
```

### 只允許區域網路訪問
```bash
python3 -m http.server 8888 --bind 192.168.1.100
```

### 背景執行
```bash
nohup python3 -m http.server 8888 --bind 0.0.0.0 &
```

### 查看正在運行的伺服器
```bash
lsof -i :8888
```

### 強制停止伺服器
```bash
kill -9 $(lsof -t -i:8888)
```
