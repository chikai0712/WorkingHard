# 🎨 繪圖畫板 - Drawing Board

一個功能完整的線上繪圖網站，支援多種繪圖工具和功能。

## ✨ 功能特色

### 繪圖工具
- ✏️ **畫筆** - 自由繪圖
- 🧹 **橡皮擦** - 清除繪圖
- 📏 **直線** - 繪製直線
- ⬜ **矩形** - 繪製矩形
- ⭕ **圓形** - 繪製圓形

### 工具選項
- 🎨 **顏色選擇** - 自訂顏色 + 8 種預設顏色
- 📏 **筆刷大小** - 1-50 像素可調
- 👁️ **透明度** - 10%-100% 可調
- ✏️ **多種畫筆** - 實心、虛線、點線、噴槍、標記筆
- 🔍 **縮放功能** - 25%-500% 可調，支援滑鼠滾輪

### 編輯功能
- ↩️ **撤銷** - 還原上一步操作（Ctrl+Z / Cmd+Z）
- ↪️ **重做** - 重做操作（Ctrl+Y / Cmd+Y）
- 🗑️ **清除** - 清除整個畫布
- 💾 **保存圖片** - 下載 PNG 格式圖片

### 其他特性
- 📱 **響應式設計** - 支援桌面和行動裝置
- 👆 **觸控支援** - 支援觸控螢幕繪圖
- ⌨️ **鍵盤快捷鍵** - 快速操作（Ctrl+Z 撤銷，Ctrl+Y 重做）
- 💾 **歷史記錄** - 最多保存 50 步操作
- 🔍 **圖片縮放** - 放大縮小檢視，支援滑鼠滾輪（Ctrl + 滾輪）

## 🚀 使用方法

### 本地運行

1. 打開 `index.html` 檔案即可使用

或使用本地伺服器：

```bash
# 使用 Python
cd Game
python3 -m http.server 8000

# 使用 Node.js (如果有安裝)
npx http-server -p 8000
```

2. 在瀏覽器中打開 `http://localhost:8000`

### 繪圖操作

1. **選擇工具** - 點擊左側工具列的工具按鈕
2. **調整顏色** - 使用顏色選擇器或點擊預設顏色
3. **調整筆刷** - 拖動筆刷大小滑桿
4. **開始繪圖** - 在畫布上按住滑鼠拖動
5. **保存作品** - 點擊「保存圖片」按鈕

### 快捷鍵

- `Ctrl+Z` / `Cmd+Z` - 撤銷
- `Ctrl+Y` / `Cmd+Y` - 重做

## 🎨 技術特點

- **HTML5 Canvas** - 使用 Canvas API 進行繪圖
- **純 JavaScript** - 無需額外套件
- **響應式設計** - CSS Flexbox 和 Grid 布局
- **現代 UI** - 漸層背景、毛玻璃效果、平滑動畫
- **觸控支援** - 完整的觸控事件處理

## 🔧 自訂選項

可以在 `script.js` 中調整：

- `MAX_HISTORY` - 歷史記錄最大數量（預設 50）
- Canvas 預設大小
- 預設顏色和筆刷大小

## 📥 下載著色圖片

網站提供了從網上下載著色圖片的功能：

### 使用 Python 爬蟲下載

1. **安裝依賴**
   ```bash
   pip3 install -r download_requirements.txt
   ```

2. **運行下載腳本**
   ```bash
   python3 download_coloring_images.py
   ```

3. **詳細說明**
   - 查看 `下載說明.md` 了解完整使用方法
   - 注意版權和使用條款

### 手動下載

- 從免費網站下載著色圖（PNG 格式推薦）
- 使用網站的「上傳本地圖片」功能載入
