# 📚 著色圖片資源說明

## 功能說明

網站支援兩種方式載入著色圖片：

### 1. 上傳本地圖片（推薦）⭐

**最簡單可靠的方法**：
- 點擊「📤 上傳本地圖片」按鈕
- 選擇您電腦中的圖片檔案
- 圖片會自動載入到畫布
- 立即開始著色

**優點**：
- ✅ 不受網路限制
- ✅ 不受跨域限制
- ✅ 可以使用任何圖片
- ✅ 支援所有常見圖片格式（PNG, JPG, GIF, SVG 等）

### 2. 線上圖片庫

目前圖片庫因為跨域（CORS）限制，線上圖片可能無法直接載入。

## 推薦的著色圖來源

### 免費著色圖網站

1. **SuperColoring** - https://www.supercoloring.com/
   - 大量免費著色圖
   - 分類清晰（動物、花卉、卡通等）
   - 支援 PNG 下載

2. **Crayola** - https://www.crayola.com/free-coloring-pages/
   - 官方免費著色頁
   - 高品質圖片
   - 適合列印

3. **OpenClipart** - https://openclipart.org/
   - 免費向量圖
   - 可下載 SVG 或 PNG
   - 商業使用友好

4. **Pixabay** - https://pixabay.com/
   - 搜尋 "coloring page" 或 "line art"
   - 免費下載
   - 高解析度圖片

### 如何下載和使用

1. 從上述網站下載著色圖（PNG 格式推薦）
2. 儲存到您的電腦
3. 在網站上點擊「📤 上傳本地圖片」
4. 選擇下載的圖片
5. 開始著色！

## 自訂圖片庫（進階）

如果您想添加自己的圖片庫，可以編輯 `images.json` 檔案：

```json
{
  "categories": [
    {
      "name": "我的圖片",
      "images": [
        {
          "id": "my_image_1",
          "name": "圖片名稱",
          "url": "圖片完整URL或本地路徑",
          "thumbnail": "縮圖URL（可選）"
        }
      ]
    }
  ]
}
```

**注意**：由於瀏覽器安全限制，本地檔案路徑（如 `file:///`）無法直接使用，建議：
- 使用線上圖片 URL（需支援 CORS）
- 或使用上傳功能載入本地圖片

## 建議的工作流程

1. **準備圖片**
   - 從免費網站下載著色圖
   - 或自己製作線稿圖片
   - 儲存為 PNG 格式（透明背景最佳）

2. **載入圖片**
   - 使用「上傳本地圖片」功能
   - 或從圖片庫選擇（如果可用）

3. **開始著色**
   - 選擇畫筆工具
   - 選擇顏色
   - 調整筆刷大小
   - 開始創作！

4. **保存作品**
   - 點擊「保存圖片」下載完成的作品

## 常見問題

**Q: 為什麼線上圖片無法載入？**
A: 瀏覽器的 CORS（跨域資源共享）安全政策限制了跨域圖片載入。建議使用上傳功能載入本地圖片。

**Q: 支援哪些圖片格式？**
A: 支援所有常見圖片格式：PNG, JPG, JPEG, GIF, WebP, SVG 等。

**Q: 圖片會上傳到伺服器嗎？**
A: 不會。所有處理都在您的瀏覽器中本地完成，圖片不會上傳到任何伺服器，保護您的隱私。

**Q: 可以調整圖片大小嗎？**
A: 圖片會自動縮放以適應畫布大小，保持原始比例。
