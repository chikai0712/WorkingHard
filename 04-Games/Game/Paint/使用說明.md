# 🎨 著色圖片庫使用說明

## 功能說明

網站現在支援從圖片庫選擇著色圖片，載入後可以在畫布上進行著色。

## 使用方法

### 1. 打開圖片庫
點擊右上角的「📚 圖片庫」按鈕，會彈出圖片選擇視窗。

### 2. 選擇圖片
- 瀏覽不同分類的圖片（動物、花卉、卡通人物、幾何圖案等）
- 使用搜尋框搜尋特定圖片
- 使用分類下拉選單過濾圖片
- 點擊圖片進行選擇（選中的圖片會有藍色邊框和✓標記）

### 3. 載入圖片
點擊「載入選中圖片」按鈕，圖片會自動載入到畫布上並居中顯示。

### 4. 開始著色
- 選擇畫筆工具
- 選擇顏色
- 調整筆刷大小
- 在圖片上進行著色

## 圖片來源

目前圖片來自 SuperColoring.com 等免費著色圖資源網站。

## 自訂圖片

您也可以修改 `images.json` 檔案來添加自己的圖片：

```json
{
  "categories": [
    {
      "name": "自訂分類",
      "images": [
        {
          "id": "custom_1",
          "name": "圖片名稱",
          "url": "圖片完整URL",
          "thumbnail": "縮圖URL（可選）"
        }
      ]
    }
  ]
}
```

## 注意事項

- 圖片需要支援 CORS（跨域資源共享）才能正常載入
- 如果圖片無法載入，請檢查圖片 URL 是否正確
- 建議使用 PNG 格式的透明背景圖片，效果最佳

## 故障排除

如果圖片無法載入：
1. 檢查網路連接
2. 確認圖片 URL 是否有效
3. 某些網站可能不允許跨域載入，需要將圖片下載到本地
4. 可以使用瀏覽器開發工具（F12）查看具體錯誤訊息
