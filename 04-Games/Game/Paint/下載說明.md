# 📥 著色圖片下載工具使用說明

## 功能說明

這個工具可以幫助您從 lovepik.com 等網站批量下載著色圖片，然後在繪圖網站中使用。

## 安裝依賴

```bash
cd /Users/ckchiu/Desktop/Project/Game
pip3 install -r download_requirements.txt
```

或者手動安裝：

```bash
pip3 install requests beautifulsoup4 lxml
```

## 使用方法

### 基本使用

```bash
cd /Users/ckchiu/Desktop/Project/Game
python3 download_coloring_images.py
```

### 使用步驟

1. **運行腳本**
   ```bash
   python3 download_coloring_images.py
   ```

2. **輸入下載數量**
   - 腳本會詢問要下載多少張圖片
   - 直接按 Enter 使用預設值（10張）

3. **等待下載完成**
   - 圖片會保存到 `downloaded_images/` 資料夾
   - 元數據會保存到 `downloaded_images/downloaded_metadata.json`

4. **在網站中使用**
   - 打開 `index.html`
   - 點擊「📚 圖片庫」
   - 點擊「📤 上傳本地圖片」
   - 選擇下載的圖片開始著色

## 注意事項

### ⚠️ 版權和使用條款

1. **僅供個人學習使用**
   - 下載的圖片僅供個人學習和練習使用
   - 請勿用於商業用途

2. **遵守網站服務條款**
   - 請遵守 lovepik.com 的使用條款
   - 尊重原作者的版權

3. **合理使用**
   - 腳本已設置下載間隔，避免對服務器造成壓力
   - 建議一次下載數量不要過多

### 🔧 技術限制

1. **網站結構變更**
   - 如果網站更新了 HTML 結構，爬蟲可能失效
   - 需要根據實際情況調整選擇器

2. **圖片格式**
   - 優先下載 PNG 格式的圖片（透明背景最佳）
   - 其他格式（JPG、GIF 等）也可以使用

3. **圖片品質**
   - 下載的是網頁顯示的圖片，可能不是最高品質
   - 如需高品質圖片，建議直接在網站付費下載

## 自訂下載

### 修改目標 URL

編輯 `download_coloring_images.py`，修改 `url` 變數：

```python
url = "https://zh.lovepik.com/images/coloring.html"  # 改為您想要的頁面
```

### 調整選擇器

如果網站結構不同，需要調整 BeautifulSoup 的選擇器：

```python
# 在 scrape_lovepik_coloring 方法中
img_tags = soup.find_all('img', src=True)  # 調整這裡
```

### 修改保存路徑

```python
downloader = ColoringImageDownloader(download_dir="my_images")  # 自訂資料夾名稱
```

## 其他下載方式

### 方法 1: 手動下載

3. 使用網站的「上傳本地圖片」功能

- **Batch Image Downloader**

### 方法 3: 其他免費資源網站

推薦的免費著色圖網站（可直接下載）：

1. **SuperColoring** - https://www.supercoloring.com/
   - 大量免費著色圖
   - 可直接下載 PNG

2. **Crayola** - https://www.crayola.com/free-coloring-pages/
   - 官方免費著色頁
   - 高品質圖片

3. **OpenClipart** - https://openclipart.org/

   - 可下載 SVG 或 PNG

4. **Pixabay** - https://pixabay.com/
   - 搜尋 "coloring page" 或 "line art"
   - 免費下載

## 故障排除

### 問題：下載失敗或找不到圖片

**可能原因：**
- 網站結構已變更
- 網路連接問題
- 網站反爬蟲機制

**解決方法：**
1. 檢查網路連接
2. 嘗試手動訪問目標 URL
3. 調整爬蟲選擇器（需要查看網頁源代碼）
4. 使用瀏覽器擴展替代

### 問題：圖片品質不佳

**解決方法：**
1. 查找圖片的高品質版本 URL
2. 在網站上尋找「下載高清」按鈕
3. 使用其他提供高品質圖片的網站

### 問題：權限錯誤

**解決方法：**
```bash
# 確保有寫入權限
chmod 755 downloaded_images/
```

## 進階用法

### 批量處理多個頁面

可以修改腳本，循環處理多個 URL：

```python
urls = [
    "https://zh.lovepik.com/images/coloring.html",
    "https://zh.lovepik.com/images/coloring-animals.html",
    # 更多 URL...
]

for url in urls:
    downloaded = downloader.scrape_lovepik_coloring(url, max_images=10)
    time.sleep(2)  # 頁面之間暫停
```

### 過濾特定類型的圖片

在 `scrape_lovepik_coloring` 方法中添加過濾邏輯：

```python
# 只下載特定尺寸的圖片
if 'large' not in img_url.lower():
    continue
```
