# 🍕 三麗鷗披薩店 | Sanrio Pizza Shop

一個可愛的餐廳經營遊戲，靈感來自 Cat Pizza！玩家扮演三麗鷗角色經營披薩店，製作披薩滿足客人需求。

## 🎮 遊戲特色

- **可愛的三麗鷗角色**：大耳狗、庫洛米、布丁狗、美樂蒂
- **餐廳經營玩法**：接待客人、製作披薩、賺取金幣
- **角色解鎖系統**：用金幣解鎖新角色，獲得特殊能力
- **時間管理挑戰**：在客人耐心耗盡前完成訂單
- **每日營業系統**：每天 2 分鐘營業時間，查看每日報表

## 🎯 遊戲玩法

### 基本流程
1. **選擇客人**：點擊左側等待的客人，查看他們的訂單
2. **製作披薩**：點擊配料按鈕添加到披薩上
3. **烤披薩**：配料添加完成後，點擊「烤披薩」按鈕
4. **上菜**：披薩烤好後，點擊「上菜」按鈕服務客人

### 配料種類
- 🍅 **醬料**：基本配料
- 🧀 **起司**：基本配料
- 🍖 **臘腸**：肉類配料
- 🍄 **蘑菇**：蔬菜配料
- 🫒 **橄欖**：蔬菜配料
- 🍍 **鳳梨**：水果配料

### 評分系統
- ⭐⭐⭐ **完美**：訂單 100% 正確 → +50💰 +10⭐
- ⭐⭐ **很好**：訂單 70% 正確 → +30💰 +5⭐
- ⭐ **普通**：訂單 40% 正確 → +15💰 +0⭐
- ☆ **糟糕**：訂單錯誤太多 → +5💰 -5⭐

### 速度獎勵
- 在客人耐心條 > 80% 時完成 → 額外 +20💰 +5⭐

### 客人耐心系統
- 🟢 **綠色 (100-60%)**：客人開心
- 🟡 **黃色 (60-30%)**：客人不耐煩
- 🔴 **紅色 (30-0%)**：客人即將離開
- 客人離開 → -10⭐

## 💡 遊戲技巧

### 新手建議
1. **先看訂單**：選擇客人後仔細看他們要什麼配料
2. **基本配料**：醬料和起司是大部分訂單的基礎
3. **不要慌張**：客人耐心條還很長時，慢慢製作
4. **使用重做**：做錯了可以點「重做」按鈕重新開始

### 進階技巧
1. **優先服務**：先服務耐心條快沒的客人
2. **角色切換**：根據情況切換角色（賺錢用庫洛米，趕時間用大耳狗）
3. **速度獎勵**：盡量在客人耐心 > 80% 時完成，獲得額外獎勵
4. **解鎖順序**：建議先解鎖布丁狗（新手友好），再解鎖美樂蒂
