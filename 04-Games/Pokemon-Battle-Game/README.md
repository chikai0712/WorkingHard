# 🎮 寶可夢對戰遊戲

**小智 vs 電腦：黑天噴火龍之戰**

一個使用 Python 和 Pygame 開發的回合制寶可夢對戰遊戲。

---

## 📋 遊戲特色

- ✅ **3 vs 3 對戰**：小智與電腦各有 3 隻寶可夢
- ✅ **回合制戰鬥**：玩家攻擊後，電腦自動反擊
- ✅ **靈活換人**：隨時可以切換寶可夢
- ✅ **強制換人**：寶可夢倒下時必須選擇下一隻
- ✅ **精美 UI**：血條、按鈕、訊息提示
- ✅ **背景音樂**：Charizard's Black Sky 主題曲

## 🚀 快速開始

### 1. 安裝依賴

```bash
pip install -r requirements.txt
```

或直接安裝 Pygame：

```bash
pip install pygame
```

### 2. 準備資源文件

創建以下目錄結構並放入對應的圖片和音樂文件：

```
Pokemon-Battle-Game/
├── assets/
│   ├── images/
│   │   ├── background.png      # 背景圖片
│   │   ├── charizard.png       # 噴火龍
│   │   ├── blastoise.png       # 水箭龜
│   │   ├── venusaur.png        # 妙蛙花
│   │   ├── garchomp.png        # 烈咬陸鯊
│   │   ├── kingdra.png         # 刺龍王
│   │   └── persian.png         # 貓老大
│   └── audio/
│       └── bgm.mp3             # 背景音樂
```

**注意**：如果沒有圖片或音樂文件，遊戲仍可運行，但會使用預設的灰色方塊代替圖片，且沒有背景音樂。

### 3. 執行遊戲

```bash
python main.py
```

## 🎮 操作說明

### 滑鼠操作
- **點擊畫面**：發動攻擊
- **點擊下方按鈕**：切換寶可夢

### 鍵盤快捷鍵
- **1/2/3**：快速切換到第 1/2/3 隻寶可夢
- **空白鍵**：發動攻擊
- **ESC**：退出遊戲

## 📖 遊戲規則

1. **回合制戰鬥**
   - 玩家點擊攻擊後，先對電腦造成傷害
   - 電腦自動反擊，對玩家造成傷害

2. **換人機制**
   - **主動換人**：隨時可以點擊下方按鈕切換寶可夢
   - **強制換人**：當前寶可夢倒下時，必須選擇下一隻才能繼續

3. **勝負判定**
   - 當一方所有寶可夢都失去戰鬥能力時，遊戲結束
   - 小智全隊倒下 → 電腦獲勝
   - 電腦全隊倒下 → 小智獲勝

## 🔧 技術架構

## 🐛 常見問題

### Q1: 圖片無法載入怎麼辦？
**A**: 確認以下事項：
- 圖片文件放在正確的路徑（`assets/images/`）
- 圖片格式為 PNG
- 檔案名稱正確（區分大小寫）

如果沒有圖片，遊戲會使用灰色方塊代替，仍可正常遊玩。

### Q2: 音樂無法播放怎麼辦？
**A**: 嘗試以下方法：
- 確認音樂文件在 `assets/audio/bgm.mp3`
- 嘗試將 MP3 轉換為 WAV 格式
- 檢查音量設定

### Q3: 中文顯示亂碼怎麼辦？
**A**: 程式會自動嘗試載入微軟雅黑字體，如果系統沒有該字體，會使用預設字體。

### Q4: 遊戲運行緩慢怎麼辦？
**A**: 可以調整以下參數：
- 降低圖片解析度
- 調整 FPS（在 `main.py` 中修改 `FPS` 變數）
