# ⚡ Phaser 3 版本說明

## 📖 概述

已創建一個基於 **Phaser 3** 的遊戲版本，包含《荒野亂鬥》的核心機制：
- ✅ WASD 移動控制
- ✅ 滑鼠指向射擊
- ✅ **三格彈藥系統**（自動回復）
- ✅ 1.5 秒回一格彈藥
- ✅ 物理碰撞系統
- ✅ 敵人 AI

## 🎮 核心機制

### 1. 彈藥系統

#### 三格彈藥限制
- 最多同時擁有 3 發子彈
- 每次射擊消耗 1 發彈藥
- 彈藥用盡後無法射擊

#### 自動回復機制
- **回復時間**：每 1.5 秒回復 1 發彈藥
- **視覺反饋**：彈藥條顯示當前彈藥和裝彈進度
- **裝彈動畫**：正在裝彈的格子會有脈衝動畫

#### UI 顯示
```
彈藥條: [●] [●] [●]  ← 三發彈藥
        [●] [●] [ ]  ← 兩發彈藥（第三發正在裝彈）
        [●] [ ] [ ]  ← 一發彈藥
```

### 2. 移動系統

- **WASD** 控制方向
- 平滑的移動體驗
- 世界邊界碰撞
- 可變移動速度（根據角色屬性）

### 3. 射擊系統

- **滑鼠點擊** 發射子彈
- **指向性射擊**：子彈朝向滑鼠位置
- **即時攻擊**：點擊即發射
- **彈藥限制**：需要彈藥才能射擊

### 4. 戰鬥系統

- **生命值系統**：顯示當前生命值
- **傷害計算**：攻擊力 - 防禦力
- **視覺反饋**：受到傷害時變紅
- **勝利條件**：擊敗敵人

## 🎯 與原版的差異

| 功能 | Canvas 版本 | Phaser 3 版本 |
|------|------------|--------------|
| 框架 | 原生 Canvas API | Phaser 3 引擎 |
| 彈藥系統 | ❌ 無 | ✅ 三格彈藥限制 |
| 自動回復 | ❌ 無 | ✅ 1.5秒回一格 |
| 物理系統 | 手動實現 | ✅ Arcade Physics |
| 碰撞檢測 | 手動計算 | ✅ 自動碰撞檢測 |
| 性能 | 中等 | ✅ 優化更好 |
| 擴展性 | 中等 | ✅ 易於擴展 |

## 🚀 使用方法

### 方法 1：直接打開
1. 在主界面點擊「⚡ Phaser 版本」按鈕
2. 或直接打開 `game-phaser.html`

### 方法 2：本地伺服器
```bash
cd BrawlStars
python3 -m http.server 8000
# 訪問 http://localhost:8000/game-phaser.html
```
