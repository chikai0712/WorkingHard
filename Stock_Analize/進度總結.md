## 🔧 技術架構

### 後端
- **框架**: FastAPI
- **資料庫**: SQLite (開發環境)
- **ORM**: SQLAlchemy
- **定時任務**: APScheduler
- **網頁爬蟲**: BeautifulSoup, requests

### 前端
- **框架**: Vue 3
- **構建工具**: Vite
- **UI 框架**: Tailwind CSS
- **HTTP 客戶端**: Axios
- **狀態管理**: Pinia
- **路由**: Vue Router

### 資料抓取策略
1. **優先使用官方 API** (TWSE, TAIFEX)
2. **備用網頁爬蟲** (HiStock)
3. **自動切換機制** (主要來源失敗時自動使用備用來源)

## 📝 使用說明

### 手動執行資料抓取
```bash
cd backend
source venv/bin/activate
python scripts/fetch_futures_data.py
```

### 啟動後端服務
```bash
cd backend
source venv/bin/activate
uvicorn main:app --reload
```

### 啟動前端服務
```bash
cd frontend
npm install
npm run dev
```
