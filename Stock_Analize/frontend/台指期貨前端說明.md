# 台指期貨前端頁面說明

## 📋 已建立的組件

### 1. 主要頁面

- ✅ **FuturesDashboard.vue** - 台指期貨儀表板主頁面
  - 位置：`src/views/FuturesDashboard.vue`
  - 路由：`/futures`
  - 功能：整合所有期貨資料組件，自動更新資料

### 2. 功能組件

#### ✅ InstitutionalTradingCard.vue
- 三大法人買賣超卡片
- 顯示外資、投信、自營商的買進、賣出、淨買賣超
- 包含自營商細分（自買、避險）
- 顯示三大法人合計

#### ✅ MarginTradingCard.vue
- 融資融券餘額卡片
- 顯示融資餘額（億元）、融券餘額（張）、借券賣出（張）
- 顯示相關指數價格和漲跌幅

#### ✅ OpenInterestCard.vue
- 期貨未平倉量卡片
- 顯示三大法人未平倉口數
- 顯示十大交易人、十大特定人、散戶未平倉

#### ✅ OptionsStrikeChart.vue
- 選擇權履約價分布圖
- 橫向條狀圖顯示買權OI（紅色，左側）和賣權OI（綠色，右側）
- 支援周選擇權和月選擇權
- 自動計算條狀圖寬度比例
- 標示價平履約價（藍色背景）

#### ✅ FuturesHistoryTable.vue
- 期貨歷史資料表格
- 顯示近30日的交易資料
- 包含指數、漲跌、漲跌幅、成交量、未平倉量

---

## 🎨 UI 設計特點

### 顏色規範
- **紅色**：上漲、正數、買權OI
- **綠色**：下跌、負數、賣權OI
- **藍色**：價平標示、十大交易人
- **紫色**：十大特定人
- **灰色**：背景、中性資料

### 響應式設計
- 使用 Tailwind CSS Grid 系統
- 支援桌面和行動裝置
- 表格可橫向滾動

## 🔌 API 整合

所有 API 呼叫都在 `src/services/futuresApi.js` 中定義：

```javascript
// 期貨 API
futuresAPI.getDaily()           // 期貨每日資料
futuresAPI.getInstitutional()   // 三大法人買賣超
futuresAPI.getOpenInterest()    // 期貨未平倉量
futuresAPI.getMarginTrading()   // 融資融券餘額

// 選擇權 API
optionsAPI.getStrikeData()      // 選擇權履約價分布
optionsAPI.getOpenInterest()    // 選擇權未平倉量
optionsAPI.getDaily()           // 選擇權每日資料
```

## 📊 資料更新機制

- **自動更新**：每 30 秒自動重新載入所有資料
- **更新間隔**：可在 `.env` 設定 `VITE_UPDATE_INTERVAL_SECONDS`
- **載入狀態**：每個組件都有獨立的 loading 狀態

## 🚀 使用方式

### 1. 訪問頁面

在瀏覽器中訪問：`http://localhost:5173/futures`

### 2. 從股票儀表板導航

在股票儀表板的右上角有「台指期貨儀表板」按鈕，點擊即可跳轉。

### 3. 路由配置

路由已註冊在 `src/router/index.js`：

```javascript
{
  path: '/futures',
  name: 'FuturesDashboard',
  component: FuturesDashboard
}
```

## 📝 資料顯示說明

### 三大法人買賣超
- 數值單位：億元
- 正數（紅色）：淨買超
- 負數（綠色）：淨賣超

### 融資融券餘額
- 融資單位：億元
- 融券/借券單位：張
- 增減顯示與股票相反（正數=增加=紅色，負數=減少=綠色）

### 未平倉量
- 單位：口數
- 正數（紅色）：做多
- 負數（綠色）：做空

### 選擇權分布圖
- 左側紅色條：買權OI（看漲）
- 右側綠色條：賣權OI（看跌）
- 條狀圖寬度依比例計算
- 價平履約價會有藍色背景標示

## 🔧 自訂與擴展

### 修改更新間隔

在 `FuturesDashboard.vue` 中：

```javascript
const interval = parseInt(import.meta.env.VITE_UPDATE_INTERVAL_SECONDS || 30) * 1000
setInterval(loadAllData, interval)
```

### 添加新的資料組件

1. 在 `src/components/futures/` 建立新組件
2. 在 `FuturesDashboard.vue` 中引入並使用
3. 在 `loadAllData()` 中載入對應的 API 資料
