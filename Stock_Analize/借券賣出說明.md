# 借券賣出資料說明

## 確認結果

根據用戶確認，**借券賣出顯示為 0 是正確的**，並非資料抓取錯誤。

## 可能的原因

1. **資料來源限制**
   - TWSE 融資融券 API (`/marginTrading/MI_MARGN`) 不包含借券賣出餘額資料
   - 該 API 只提供融資、融券的統計資料

2. **實際市場狀況**
   - 當日可能確實沒有借券賣出交易
   - 或借券賣出餘額確實為 0

3. **資料結構**
   - 資料庫欄位 `securities_lending_sell` 預設值為 0
   - 當 API 不提供該資料時，會使用預設值

## 處理方式

目前系統會：
- ✅ 正確顯示 0（如果資料確實為 0）
- ✅ 正確顯示「無資料」（如果資料為 null 或 undefined）

## 前端顯示邏輯

```javascript
// 如果資料為 0，顯示 0
// 如果資料為 null/undefined，顯示「無資料」
{{ formatNumber(data.securities_lending?.sell) || 0 }} 張
{{ data.securities_lending?.change !== null ? formatChange(...) : '無資料' }}
```

## 如果需要完整的借券賣出資料

如果需要取得完整的借券賣出餘額資料，可能需要：

1. **查閱 TWSE 官方 API 文件**
   - 尋找專門的借券賣出 API 端點
   - 確認是否有其他資料來源

2. **使用網頁爬蟲**
   - 如果 API 不提供，可能需要從網頁爬取

3. **整合第三方資料來源**
   - 如果 TWSE 官方不提供，可能需要尋找其他資料來源

## 現狀

- ✅ 系統能正確處理借券賣出為 0 的情況
- ✅ 前端會正確顯示 0 或「無資料」
- ✅ 不會因為顯示 0 而誤導用戶
