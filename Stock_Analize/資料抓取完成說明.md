## ⚠️ 注意事項

1. **交易日限制**
   - API 只在交易日有資料
   - 查詢非交易日會返回空資料

2. **API 限流**
   - 建議在請求之間加入延遲（目前設定 1 秒）
   - 避免過於頻繁的請求

3. **資料格式變更**
   - API 回應格式可能變更
   - 需要定期檢查和更新解析邏輯
