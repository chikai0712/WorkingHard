# 三大法人買賣超資料來源調查

## 問題分析

目前系統中的三大法人資料存在問題：
- **外資資料**：全部為 0（缺失）
- **投信資料**：全部為 0（缺失）
- **自營資料**：有資料，但可能不完整

## 可能的原因

1. **TAIFEX 頁面結構複雜**：表格解析邏輯未能正確識別外資和投信的行
2. **資料格式變化**：TAIFEX 可能更新了頁面結構
3. **資料來源問題**：使用的 API 或頁面可能不包含完整的三大法人資料

---

## 可用的資料來源

### 1. HiStock (嗨投資) ✅

**URL**: `https://histock.tw/stock/three.aspx`

**優點**：
- 資料格式清晰
- 包含完整的三大法人資料（外資、投信、自營）
- 易於解析

**缺點**：
- 第三方網站，資料可能有延遲
- 需要網頁爬蟲

**狀態**：✅ 已實現，但需要確認資料正確性

---

### 2. TAIFEX 官方網站

**URL**: `https://www.taifex.com.tw/cht/3/futContractsDate`

**優點**：
- 官方資料來源
- 資料最權威

**缺點**：
- 頁面結構複雜，解析困難
- 目前只能解析到自營商資料

**狀態**：⚠️ 已實現，但解析不完整

---

### 3. 籌碼K線

**說明**：可能是一個股票分析網站或APP

**需要調查**：
- 是否有公開 API
- 是否提供資料下載
- 資料授權和使用條款

**狀態**：❓ 待調查

---

### 4. 富果 (Fugle) API

**URL**: `https://developer.fugle.tw/`

**需要調查**：
- 是否有免費方案
- 是否包含三大法人期貨資料
- API 授權和使用方式

**狀態**：❓ 待調查

---

### 5. 證交所/期交所 API

**可能的端點**：
- TWSE API（但主要是股票資料）

- 是否有公開 API

**狀態**：❓ 待調查

## 建議的解決方案

### 方案 1: 改進 HiStock 爬蟲（推薦）

**優點**：
- 資料格式清晰
- 已經有實現基礎
- 可以快速取得正確資料

**步驟**：
1. 確認 HiStock 的資料是否正確
2. 改進日期選擇邏輯（目前總是取得最新日期）
3. 將其設為主要資料來源

---

### 方案 2: 改進 TAIFEX 解析邏輯

**優點**：
- 官方資料來源
- 資料最權威

**步驟**：
1. 詳細分析 TAIFEX 頁面結構
2. 找出外資和投信資料的正確位置
3. 改進解析邏輯

---

### 方案 3: 使用第三方 API 服務

**可能的服務**：
- 富果 API
- 其他金融資料 API

**步驟**：
1. 註冊 API 帳號
2. 查看 API 文件
3. 實現 API 客戶端
4. 整合到系統中
