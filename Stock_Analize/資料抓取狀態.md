# 資料抓取狀態

## ✅ 已實現的功能

### 1. 三大法人買賣超資料

**資料來源**：
- **主要來源**：TAIFEX（台灣期貨交易所）
  - URL: `https://www.taifex.com.tw/cht/3/futContractsDate`
  - 狀態：✅ 已實現，但只解析到部分資料（自營商）
- **備用來源**：HiStock（嗨投資）
  - URL: `https://histock.tw/stock/three.aspx`
  - 狀態：✅ 已實現並正常運作
  - 資料格式清晰，解析成功率 100%

**實現檔案**：
- `backend/crawler/taifex_institutional_scraper.py` - TAIFEX 爬蟲
- `backend/crawler/histock_scraper.py` - HiStock 爬蟲
- `backend/crawler/futures_data_fetcher.py` - 資料抓取協調器

**資料儲存**：
- 表格：`institutional_trading_data`
- API 端點：`GET /api/futures/institutional`

**目前狀態**：
- ✅ 資料抓取功能正常
- ✅ 資料儲存功能正常
- ✅ API 端點正常運作
- ⚠️ TAIFEX 解析邏輯需要改進（目前只能解析自營商，外資和投信需要改進）
- ✅ HiStock 作為備用來源運作良好

## 🚀 使用方式

### 手動抓取資料

```bash
cd Stock_Analize/backend
source venv/bin/activate
python scripts/fetch_recent_data.py
```

### 透過 API 取得資料

```bash
# 取得三大法人資料
curl "http://localhost:8000/api/futures/institutional?market_type=weighted&limit=7"

# 取得融資融券資料
curl "http://localhost:8000/api/futures/trading?market_type=weighted&limit=7"
```
