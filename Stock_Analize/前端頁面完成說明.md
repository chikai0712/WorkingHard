## 🎨 頁面功能

### 1. 三大法人買賣超
- 顯示外資、投信、自營商的買進、賣出、淨買賣超
- 包含自營商細分（自買、避險）
- 顯示三大法人合計
- 顏色標示：正數紅色，負數綠色

### 2. 融資融券餘額
- 顯示融資餘額（億元）及增減
- 顯示融券餘額（張）及增減
- 顯示借券賣出（張）及增減
- 顯示相關指數價格和漲跌幅

### 3. 期貨未平倉量
- 顯示三大法人未平倉口數（外資、投信、自營）
- 顯示十大交易人未平倉（前五大、前十大）
- 顯示十大特定人未平倉
- 顯示散戶未平倉
- 每個項目都顯示未平倉口數和增減變化

### 4. 選擇權履約價分布圖
- **周選擇權分布圖**：顯示各履約價的買權OI（紅色，左側）和賣權OI（綠色，右側）
- **月選擇權分布圖**：同上，但為月合約
- 橫向條狀圖，寬度依比例計算
- 價平履約價會有藍色背景標示
- 顯示履約價、OI數值、增減變化

### 5. 歷史資料表格
- 顯示近30日期貨交易資料
- 包含：日期、指數、漲跌、漲跌幅、成交量、未平倉量
- 支援滾動查看

## 🔄 資料更新

- **自動更新**：每 30 秒自動重新載入所有資料
- **更新間隔**：可在 `.env` 設定 `VITE_UPDATE_INTERVAL_SECONDS`
- **即時顯示**：每個組件獨立載入，互不影響

## 📊 資料顯示說明

### 數值格式
- **億元**：使用千分位格式化，小數點後2位
- **口數/張數**：使用千分位格式化，無小數
- **百分比**：小數點後2位，帶正負號

### 顏色規範
- **紅色**：上漲、正數、買超、做多、買權OI
- **綠色**：下跌、負數、賣超、做空、賣權OI
- **藍色**：價平標示、十大交易人區域
- **紫色**：十大特定人區域
- **灰色**：背景、中性資料

## ⚠️ 注意事項

### 1. 後端 API 必須運行
前端頁面需要後端 API 提供資料，請確保：
- 後端服務正在運行：`http://localhost:8000`
- API 端點正常回應

### 2. 資料格式對應
前端組件期望的資料格式與後端 API 回傳格式必須一致。如果後端 API 尚未實現，可能會顯示「目前沒有資料」。

### 3. 選擇權分布圖
- 當履約價數量很多時，圖表會自動出現滾動條
- 條狀圖寬度會根據最大 OI 值自動調整比例
- 如果沒有資料，會顯示「目前沒有資料」提示

## 🐛 除錯建議

### 如果頁面顯示「目前沒有資料」：

1. **檢查 API 連接**
   ```javascript
   // 在瀏覽器開發者工具 Console 中測試
   fetch('http://localhost:8000/api/futures/institutional?market_type=weighted&limit=1')
     .then(r => r.json())
     .then(console.log)
   ```

2. **檢查 API 回應格式**
   - 確認後端 API 回傳的資料格式符合前端組件的預期
   - 參考 `台指期貨API說明.md` 中的回應格式

3. **檢查網路請求**
   - 打開瀏覽器開發者工具的 Network 標籤
   - 查看 API 請求是否成功
   - 查看錯誤訊息

### 如果組件無法顯示：

1. **檢查組件導入**
   - 確認所有組件檔案都已建立
   - 確認 `FuturesDashboard.vue` 中的 import 路徑正確

2. **檢查路由配置**
   - 確認 `router/index.js` 中已正確註冊路由
   - 確認路由路徑為 `/futures`
