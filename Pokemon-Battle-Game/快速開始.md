### 第 1 步：執行自動安裝（只需一次）

打開終端機，複製並執行：

```bash
cd /Users/ckchiu/Desktop/Project/Pokemon-Battle-Game
bash setup.sh
```

**這個腳本會自動**：
- 創建虛擬環境
- 安裝 Pygame
- 創建資源目錄
- 驗證安裝

**預計時間**：1-2 分鐘

### 第 2 步：運行遊戲

安裝完成後，執行：

```bash
bash run.sh
```

或手動執行：

```bash
source venv/bin/activate
python main.py
```

**遊戲操作**：
- 點擊畫面 = 攻擊

- 按 ESC = 退出

**即使沒有圖片也能玩**：
