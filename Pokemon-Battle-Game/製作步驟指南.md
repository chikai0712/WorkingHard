## 🔧 第一步：安裝 Pygame

由於系統權限問題，請您手動在終端機執行以下命令：

### 方法 1：使用 Homebrew（推薦）

```bash
# 如果還沒安裝 Homebrew，先安裝它
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 使用 Homebrew 安裝 Python 和 Pygame
brew install python3
brew install pygame
```

### 方法 2：使用 pip（需要管理員權限）

```bash
# 在終端機中執行（可能需要輸入密碼）
sudo pip3 install pygame
```

### 方法 3：使用虛擬環境（最安全）

```bash
# 進入專案目錄
cd /Users/ckchiu/Desktop/Project/Pokemon-Battle-Game

# 創建虛擬環境
python3 -m venv venv

# 啟動虛擬環境
source venv/bin/activate

# 安裝 Pygame
pip install pygame

# 之後每次運行遊戲前都要先啟動虛擬環境
# source venv/bin/activate
```

### 驗證安裝

安裝完成後，執行以下命令驗證：

```bash
python3 -c "import pygame; print(pygame.version.ver)"
```

如果顯示版本號（例如：2.5.2），表示安裝成功！

## 🎯 遊戲操作說明

### 滑鼠操作
- **點擊畫面任意處**：發動攻擊
- **點擊下方按鈕**：切換寶可夢

### 鍵盤快捷鍵
- **1/2/3**：快速切換到第 1/2/3 隻寶可夢
- **空白鍵**：發動攻擊
- **ESC**：退出遊戲

### 遊戲規則
1. 點擊攻擊後，玩家先攻擊，電腦自動反擊
2. 當寶可夢 HP 歸零時：
   - 電腦會自動換下一隻
   - 玩家必須手動選擇下一隻
3. 當一方所有寶可夢都倒下時，遊戲結束

## 🐛 常見問題排除

### 問題 1：無法安裝 Pygame

**解決方案**：
```bash
# 嘗試升級 pip
python3 -m pip install --upgrade pip --user

# 或使用 Homebrew
brew install pygame
```

### 問題 2：圖片無法載入

**檢查清單**：
- [ ] 圖片文件在 `assets/images/` 目錄
- [ ] 檔案名稱完全正確（區分大小寫）
- [ ] 圖片格式為 PNG
- [ ] 檔案沒有損壞

**測試命令**：
```bash
ls -la assets/images/
file assets/images/charizard.png
```

### 問題 3：音樂無法播放

**解決方案**：
- 確認音樂文件在 `assets/audio/bgm.mp3`
- 嘗試轉換為 WAV 格式
- 檢查音量設定

### 問題 4：視窗無法打開

**可能原因**：
- Pygame 未正確安裝
- 顯示器設定問題

**解決方案**：
```bash
# 重新安裝 Pygame
pip3 uninstall pygame
pip3 install pygame
```

### 問題 5：中文顯示亂碼

**說明**：
- 程式會自動嘗試載入微軟雅黑字體
- 如果系統沒有該字體，會使用預設字體
- 不影響遊戲運行
