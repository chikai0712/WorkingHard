# 🐕🧋 柴犬珍奶店

一個療癒系的珍珠奶茶製作遊戲，由可愛的柴犬經營，服務各種動物顧客！

## 🎮 遊戲特色

### 核心玩法
- **搖茶**：左右拖曳搖晃器，感受真實的搖茶手感
- **倒茶**：長按倒茶到目標線，考驗你的精準度
- **加珍珠**：點擊加入 Q 彈珍珠，看它們掉落的療癒動畫
- **封膜**：向右滑動完成封膜，大功告成！

### 視覺設計
- 溫暖的橙黃色系，柴犬主題配色
- 流暢的彈性動畫，每個操作都有反饋
- 可愛的動物顧客，每位都有獨特表情
- 精緻的 UI 設計，圓潤舒適的視覺體驗

### 評價系統
- ⭐⭐⭐ **完美**：4 個步驟都完美執行，獲得 50 金幣
- ⭐⭐ **很好**：2-3 個步驟完美，獲得 30 金幣
- ⭐ **不錯**：完成任務，獲得 20 金幣

## 📖 遊戲說明

### 操作方式

#### 桌面版
- **搖茶**：滑鼠拖曳搖晃器左右移動
- **倒茶**：按住滑鼠左鍵倒茶，放開停止
- **加珍珠**：點擊杯子加入珍珠
- **封膜**：按住並向右拖曳

#### 手機版
- **搖茶**：手指拖曳搖晃器左右移動
- **倒茶**：長按螢幕倒茶，放開停止
- **加珍珠**：點擊杯子加入珍珠
- **封膜**：按住並向右滑動

### 完美技巧

1. **搖茶**：剛好搖動 10 次，不多不少
2. **倒茶**：倒到綠色目標線位置（70% 高度，±5% 容錯）
3. **加珍珠**：剛好加入 8 顆珍珠
4. **封膜**：一次性滑到 100%

## 🎨 設計理念

### 為什麼選擇柴犬？
- 表情豐富，適合做反饋動畫
- 全球認知度高，跨文化吸引力
- 可愛但不幼稚，適合各年齡層

### 為什麼選擇珍奶？
- 視覺層次豐富（透明杯、茶色、珍珠）
- 操作感強（搖、倒、加料、封膜）
- 文化熱度高，亞洲和歐美都有市場
- 配料擴展性強，可持續更新內容

### 色彩心理學
- **橙色 (#FF8C42)**：溫暖、活力、食慾
- **黃色 (#FFD166)**：快樂、陽光、美味
- **綠色 (#06FFA5)**：清新、健康、平衡

## 💡 開發心得

### 成功要素
1. **專注核心**：4 個步驟，每個都做到極致
2. **即時反饋**：每個操作都有視覺和動畫反饋
3. **不設失敗**：降低挫折感，提升留存率
4. **追求完美**：給進階玩家挑戰目標

### 設計陷阱
- ❌ 不要一開始就做太多功能
- ❌ 不要忽視音效的重要性
- ❌ 不要讓玩家等待（載入、動畫）
- ❌ 不要過度懲罰失誤
