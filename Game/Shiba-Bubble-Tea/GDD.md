## 🎯 High Concept

**一句話概念：**
> 柴犬開的珍奶店，用療癒手感製作飲料，服務可愛動物顧客。

**核心賣點：**
- 🐕 **可愛主題**：柴犬 + 動物顧客 = 治癒系畫風
- 🧋 **真實手感**：搖茶、倒茶、加料、封膜 - 每個步驟都有觸感反饋
- ⭐ **完美挑戰**：不設失敗，但追求完美評價的成就感
- 💰 **輕度經營**：賺金幣解鎖新配料、新杯子、新裝飾

**目標受眾：**
- 年齡：12-35 歲
- 性別：女性為主（70%），男性（30%）
- 喜好：喜歡可愛風格、休閒遊戲、模擬經營

## 🔄 核心循環

### 主循環（90 秒/輪）

```
顧客出現 → 顯示訂單 → 製作飲料 → 獲得評價 → 賺取金幣 → 下一位顧客
    ↓                                                        ↑
    └────────────────── 累積金幣解鎖新內容 ──────────────────┘
```

### 製作流程（4 步驟）

#### 步驟 1：搖茶（Shake）
- **操作**：左右拖曳搖晃器
- **目標**：搖動 10 次
- **完美條件**：剛好 10 次（不多不少）
- **反饋**：
  - 視覺：搖晃器左右擺動、旋轉
  - 音效：咕嚕咕嚕的液體聲
  - 觸覺：震動反饋（手機）

#### 步驟 2：倒茶（Pour）
- **操作**：長按倒茶，放開停止
- **目標**：倒到目標線（70% 高度）
- **完美條件**：誤差 ±5% 以內
- **反饋**：
  - 視覺：液體上升動畫、目標線發光
  - 音效：倒水聲（音調隨高度變化）
  - 觸覺：達到目標時震動

#### 步驟 3：加珍珠（Boba）
- **操作**：點擊杯子加入珍珠
- **目標**：加入 8 顆珍珠
- **完美條件**：剛好 8 顆
- **反饋**：
  - 視覺：珍珠從上方掉落、彈跳
  - 音效：啵啵聲（每顆音調不同）
  - 觸覺：每次點擊輕微震動

#### 步驟 4：封膜（Seal）
- **操作**：向右滑動封膜
- **目標**：滑到 100%
- **完美條件**：一次滑到底
- **反饋**：
  - 視覺：封膜漸變顯示、發光效果
  - 音效：撕膜聲 + 完成鈴聲
  - 觸覺：完成時強震動

## 🎮 遊戲機制

### 難度曲線（前 10 關）

| 關卡 | 新增元素 | 難度調整 |
|------|---------|---------|
| 1 | 基礎教學 | 無時間限制 |
| 2-3 | 熟悉操作 | 容錯範圍大 |
| 4 | 解鎖新顧客（🐼） | 開始計時（90 秒） |
| 5 | 解鎖新配料（布丁） | 容錯範圍縮小 |
| 6-7 | 連續訂單 | 2 杯連做 |
| 8 | 解鎖新杯子 | 視覺升級 |
| 9 | 特殊訂單（雙倍珍珠） | 步驟變化 |
| 10 | Boss 關卡 | 3 杯連做 + 完美要求 |

### 解鎖系統

#### 配料解鎖（影響玩法）
- **Level 1**：珍珠（基礎）
- **Level 5**：布丁（點擊方式不同）
- **Level 10**：椰果（需要搖晃加入）
- **Level 15**：芋圓（需要長按煮熟）
- **Level 20**：寒天（需要切割）

#### 外觀解鎖（不影響玩法）
- **杯子**：透明杯、漸層杯、卡通杯、玻璃杯
- **店面**：木質風、現代風、日式風、夢幻風
- **柴犬裝扮**：廚師帽、蝴蝶結、墨鏡、圍裙

## 📊 經濟系統

### 金幣獲取

| 來源 | 金幣 | 條件 |
|------|------|------|
| 完美評價 | 50 | 4/4 完美步驟 |
| 很好評價 | 30 | 2-3/4 完美步驟 |
| 不錯評價 | 20 | 0-1/4 完美步驟 |
| 連續完美 | +10 | 每連續 3 次完美 |
| 每日登入 | 100 | 每天首次登入 |

### 金幣消耗

| 項目 | 價格 | 效果 |
|------|------|------|
| 新配料 | 200-500 | 解鎖新玩法 |
| 新杯子 | 100-300 | 視覺變化 |
| 店面裝飾 | 500-1000 | 背景變化 |
| 柴犬裝扮 | 150-400 | 角色外觀 |
| 加速道具 | 50 | 本局製作速度 +20% |

### 平衡設計

**目標：讓玩家每 5-10 分鐘解鎖一次新內容**

- 平均每局：30 金幣（假設 50% 完美率）
- 每分鐘：2 局 = 60 金幣
- 5 分鐘：300 金幣 → 可解鎖新杯子
- 10 分鐘：600 金幣 → 可解鎖新配料

## 🎨 美術風格

### 色彩方案

**主色調：溫暖橙黃系**
```css
--primary: #FF8C42    /* 柴犬橘色 */
--secondary: #FFD166  /* 奶茶黃色 */
--accent: #06FFA5     /* 清新綠色 */
--bg-light: #FFF8F0   /* 米白背景 */
```

**為什麼選這個配色？**
- 橙色：柴犬的代表色，溫暖親切
- 黃色：珍奶的視覺聯想，食慾感
- 綠色：清新健康，平衡暖色
- 米白：乾淨舒適，不刺眼

### 字體選擇

- **主字體**：Quicksand（圓潤、現代、易讀）
- **輔助字體**：Nunito（友善、可愛）
- **中文字體**：思源黑體 / 微軟正黑體

### 動畫原則

1. **彈性動畫**：使用 `cubic-bezier(0.34, 1.56, 0.64, 1)` 製造 Q 彈感
2. **分層動畫**：不同元素錯開 0.1-0.2 秒，避免同時動
3. **微互動**：hover 時放大 1.05 倍，點擊時縮小 0.95 倍
4. **持續動畫**：柴犬、金幣等元素保持輕微晃動

    if (sounds[name]) {
        sounds[name].play();
    }

## 💻 技術規格

### 推薦技術棧

**選項 1：純 Web（最推薦）**
- HTML5 + CSS3 + Vanilla JavaScript
- 優點：無需編譯、跨平台、檔案小
- 適合：Poki、CrazyGames 等 Web 平台

**選項 2：Cocos Creator**
- 優點：內建物理引擎、動畫編輯器
- 適合：需要複雜動畫的版本

**選項 3：Godot Engine**
- 優點：開源、輕量、支援 Web 導出
- 適合：想要快速迭代的團隊

### 效能目標

| 指標 | 目標值 | 說明 |
|------|--------|------|
| 首次載入 | < 3 秒 | 包含所有資源 |
| 檔案大小 | < 5 MB | 壓縮後 |
| FPS | 60 | 流暢動畫 |
| 記憶體 | < 100 MB | 手機可運行 |

### 瀏覽器支援

- Chrome 90+
- Safari 14+
- Firefox 88+
- Edge 90+
- 手機瀏覽器（iOS Safari、Chrome Mobile）
