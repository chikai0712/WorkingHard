# 🐱🍕 貓咪披薩店

一個可愛的經營模擬遊戲，靈感來自 Cat Pizza。經營你的披薩店，雇用貓咪員工，服務可愛的動物顧客！

## 🎮 遊戲特色

- **可愛的視覺設計**：採用溫暖的粉色系配色，搭配可愛的動物角色
- **關卡系統**：通過服務顧客獲得經驗值，不斷升級解鎖新內容
- **經營管理**：賺取金錢，購買升級，擴展你的披薩店
- **自動化系統**：雇用貓咪員工自動服務顧客
- **流暢動畫**：豐富的動畫效果和互動反饋

## 🎯 遊戲玩法

### 基礎操作
1. **購買收銀台**（💵 50）- 開始營業的第一步
2. **點擊顧客**服務他們，獲得金錢和經驗值
3. **打開商店**購買升級項目

### 升級項目
- **🐱 貓咪員工**（💵 100）- 自動服務顧客，價格遞增
- **🔥 升級烤箱**（💵 150）- 加快製作速度，價格遞增
- **✨ 店面裝飾**（💵 200）- 吸引更多顧客，縮短顧客到來間隔

### 顧客類型
- 🐱 貓咪：1 個披薩，支付 💵 10
- 🐶 狗狗：2 個披薩，支付 💵 20
- 🐰 兔子：3 個披薩，支付 💵 30
- 🐻 熊熊：4 個披薩，支付 💵 40
- 🦊 狐狸：2 個披薩，支付 💵 25

### 升級系統
- 每服務一個披薩獲得 10 經驗值
- 升級後獲得 💵 50 獎勵
- 顧客到來速度加快
- 經驗需求遞增（1.5 倍）

## 🚀 如何開始

### 方法一：直接開啟
```bash
# 在瀏覽器中打開
open index.html
```

### 方法二：使用本地伺服器
```bash
# 使用 Python
python3 -m http.server 8000

# 使用 Node.js
npx http-server -p 8000

# 然後在瀏覽器訪問
# http://localhost:8000
```

## 🎯 遊戲策略

### 初期策略
1. 先存錢購買收銀台（💵 50）
2. 手動點擊服務顧客賺取初始資金
3. 優先購買第一個貓咪員工實現自動化

### 中期策略
1. 持續雇用更多員工
2. 升級烤箱提高效率
3. 平衡員工數量和烤箱等級

### 後期策略
1. 購買店面裝飾增加顧客流量
2. 最大化自動化收益
3. 快速升級獲得更多獎勵
