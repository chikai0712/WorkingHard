# 🎮 寶可夢史詩聯手對戰系統

## 📖 遊戲簡介

這是一款基於文字介面的寶可夢對戰遊戲，特色是小智與阪木老大的史詩聯手對戰系統！

## ✨ 遊戲特色

1. **🎵 背景音樂聯動**：執行即自動開啟 YouTube 音樂連結，播放《寶可夢 旅途》主題曲 - 1・2・3
2. **🔄 場景即時變換**：切換寶可夢時，背景會從火山變成科技場或森林
3. **👥 雙人聯手感**：在大招中加入了小智與阪木老大的帥氣台詞
4. **💯 滿血系統**：初始血量為 1000/1000
5. **🛡️ 防護罩系統**：藍鑽神鳥防護罩可阻擋 70% 傷害

## 🚀 執行方式

### Pygame 遊戲界面版本 ⭐⭐⭐ 最推薦
```bash
# 先安裝 pygame
pip3 install pygame

# 執行遊戲
python3 game_pygame.py
```
**特色：**
- 🎮 **真正的遊戲視窗**：獨立視窗界面，非終端機
- 🖱️ **滑鼠操作**：點擊按鈕進行遊戲
- ⌨️ **鍵盤快捷鍵**：支援 1-4, m, ESC
- 🎨 **彩色界面**：根據場景變換背景顏色
- 💫 **流暢動畫**：60 FPS 流暢體驗
- 🎵 **背景音樂**：一鍵開啟 YouTube 音樂

### 彩色終端版本 ⭐ 推薦（無需額外安裝）
```bash
python3 game_color.py
```
**特色：**
- 🎨 彩色終端界面，使用 ANSI 顏色代碼
- 📦 無需安裝任何額外套件
- 🎯 清晰的視覺化顯示
- 💫 更好的遊戲體驗

### 文字介面版本（基本版）
```bash
python3 game.py
```

### 圖形介面版本（GUI）- macOS 13.0+ 可能需要更新
```bash
python3 game_gui.py
```
**注意：** 某些 macOS 版本可能不支援，如果遇到錯誤請使用彩色終端版本。

### Rich 版本（需要安裝 rich 套件）
```bash
pip3 install rich
python3 game_rich.py
```

## 🎯 遊戲操作

- **指令 1**：使用小招/普通招式攻擊
- **指令 2**：使用特殊大招（100% 能量）
- **指令 3**：切換寶可夢（可選擇噴火龍、水箭龜、妙蛙花）
- **指令 4**：啟動藍鑽神鳥防護罩
- **q**：退出遊戲

## 📋 系統需求

- Python 3.x
- 網路連線（用於開啟 YouTube 音樂連結）
- 網頁瀏覽器（用於播放背景音樂）

## 🎨 遊戲場景

- **火山地帶**：噴火龍的主場
- **科技訓練場**：水箭龜的主場
- **月夜森林**：妙蛙花的主場

## 📝 注意事項

- 遊戲會自動開啟瀏覽器播放背景音樂
- 防護罩有使用次數限制（初始為 2 次）
- 所有寶可夢初始血量均為 1000/1000

享受小智與阪木老大聯手的史詩對戰，體驗震撼的視覺效果和精彩的戰鬥場面！
