# 遊戲引擎安裝說明

## Pygame 版本（推薦）⭐

Pygame 是最適合 Python 遊戲開發的庫，提供完整的遊戲界面功能。

### 安裝方式

```bash
# 使用 pip 安裝
pip3 install pygame

# 或使用 pip3
pip3 install -r requirements.txt
```

### 執行遊戲

```bash
python3 game_pygame.py
```

## 遊戲特色

- 🎮 **真正的遊戲視窗**：獨立視窗，非終端機
- 🖱️ **滑鼠操作**：點擊按鈕進行遊戲
- ⌨️ **鍵盤快捷鍵**：支援 1-4, m, ESC 快捷鍵
- 🎨 **彩色界面**：根據場景變換背景顏色
- 💫 **流暢動畫**：60 FPS 流暢體驗
- 🎵 **背景音樂**：一鍵開啟 YouTube 音樂

## 操作說明

### 滑鼠操作
- 點擊按鈕執行對應動作
- 滑鼠懸停按鈕會有顏色變化

### 鍵盤操作
- `1` - 小招攻擊
- `2` - 特殊大招
- `3` - 切換寶可夢
- `4` - 啟動防護罩
- `m` - 開啟背景音樂
- `ESC` - 退出遊戲

## 其他版本


### 基本文字版本
