- 連線工具：Azure Data Studio 或 DBeaver

## 工具速查

```bash
# 啟動 MSSQL
cd docker && docker compose up -d

# 停止 MSSQL
cd docker && docker compose down

# 進入 sqlcmd
bash docker/connect.sh

# 查看 container log
docker logs mssql-learning
```
