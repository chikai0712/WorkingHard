# 流量費用精簡版（20k QPS 環境）

| QPS | 出站流量 (GB/月) | AWS 流量費用 | GCP 流量費用 | AWS 總計 | GCP 總計 |

---

## 20k QPS 環境詳細費用

### AWS
- **出站流量**：4,147 GB/月
- **流量費用**：$373 / 月
- **運算+磁碟**：$1,119 / 月
- **總計**：**$1,492 / 月**

### GCP
- **出站流量**：4,147 GB/月
- **流量費用**：$466 / 月
- **運算+磁碟**：$1,232 / 月
- **總計**：**$1,698 / 月**

---

## 流量計算公式

```
出站流量 (GB/月) = QPS × 80 bytes × 86,400 秒 × 30 天 / 1024³
```

---

## 價格說明

- **AWS**：前 10TB $0.09/GB（階梯價格）
- **GCP**：前 1TB $0.12/GB，1-10TB $0.11/GB（階梯價格）
- **總計**：運算費用 + 磁碟費用 + 流量費用
