## 2. DNS 請求處理流程

```mermaid
sequenceDiagram
    participant Client as 客戶端
    participant NLB as AWS NLB
    participant K8sSvc as K8s Service
    participant DNSdist as dnsdist
    participant Unbound as Unbound
    participant AuthDNS as 權威 DNS
    participant Cache as 快取層

    Client->>NLB: 1. DNS Query (UDP/TCP)
    Note over NLB: 添加 Proxy Protocol v2 Header<br/>(包含真實客戶端 IP)

    NLB->>K8sSvc: 2. 轉發請求 (PPv2)
    Note over K8sSvc: IP Target Mode<br/>直接路由到 Pod IP

    K8sSvc->>DNSdist: 3. 請求到達 Pod

    DNSdist->>DNSdist: 4. 解析 Proxy Protocol<br/>提取真實 IP

    DNSdist->>DNSdist: 5. ACL 檢查
    alt ACL 拒絕
        DNSdist->>Client: 返回拒絕
    end

    DNSdist->>DNSdist: 6. 動態限流檢查
    alt 超過速率限制
        DNSdist->>DNSdist: 動態封鎖 IP
        DNSdist->>Client: 返回拒絕
    end

    DNSdist->>DNSdist: 7. 檢查 Packet Cache
    alt 快取命中
        DNSdist->>Client: 8a. 直接返回快取回應
    else 快取未命中
        DNSdist->>Unbound: 8b. 轉發查詢 (PPv2)

        Unbound->>Unbound: 9. 檢查本地快取
        alt 本地快取命中
            Unbound->>DNSdist: 10a. 返回快取回應
        else 本地快取未命中
            Unbound->>Unbound: 10b. 檢查過期快取<br/>(RFC 8767)

            alt 有過期快取且查詢超時
                Unbound->>DNSdist: 10c. 返回過期快取<br/>(TTL=30s)
            else 無過期快取或查詢正常
                Unbound->>AuthDNS: 11. 遞迴查詢
                AuthDNS->>Unbound: 12. DNS 回應
                Unbound->>Unbound: 13. 更新快取
                Unbound->>DNSdist: 14. 返回回應
            end
        end

        DNSdist->>DNSdist: 15. 更新 Packet Cache
        DNSdist->>Client: 16. 最終回應
    end

    Note over DNSdist,Unbound: 同時發送 DNSTAP 日誌到 Vector Sidecar
```

**Water Torture 攻擊防護流程**

```mermaid
sequenceDiagram
    participant Attacker as 攻擊者
    participant DNSdist as dnsdist
    participant Unbound as Unbound
    participant AuthDNS as 權威 DNS
    
    Note over Attacker: 發送大量隨機子域名查詢
    
    loop 攻擊階段
        Attacker->>DNSdist: 查詢 random1.example.com
        DNSdist->>Unbound: 轉發查詢
        Unbound->>AuthDNS: 遞迴查詢
        AuthDNS-->>Unbound: SERVFAIL (域名不存在)
        Unbound-->>DNSdist: SERVFAIL
        DNSdist->>DNSdist: 記錄 SERVFAIL 到 Ring Buffer
        
        Attacker->>DNSdist: 查詢 random2.example.com
        DNSdist->>Unbound: 轉發查詢
        Unbound->>AuthDNS: 遞迴查詢
        AuthDNS-->>Unbound: SERVFAIL
        Unbound-->>DNSdist: SERVFAIL
        DNSdist->>DNSdist: 記錄 SERVFAIL
    end
    
    Note over DNSdist: 分析 Ring Buffer<br/>檢測到 >50 SERVFAIL/10s
    
    DNSdist->>DNSdist: 觸發動態封鎖規則
    DNSdist->>DNSdist: 封鎖攻擊者 IP (60秒)
    
    Attacker->>DNSdist: 後續查詢
    DNSdist->>Attacker: 拒絕回應 (已封鎖)
    
    Note over DNSdist: 60秒後自動解除封鎖
```

**RFC 8767 Serve Stale 詳細流程**

```mermaid
sequenceDiagram
    participant Client as 客戶端
    participant Unbound as Unbound
    participant AuthDNS as 權威 DNS
    participant Cache as 快取
    
    Client->>Unbound: DNS 查詢 example.com
    
    Unbound->>Cache: 檢查快取
    Cache-->>Unbound: 快取已過期 (TTL=0)
    
    Note over Unbound: serve-expired-client-timeout: 1800ms
    
    Unbound->>AuthDNS: 嘗試查詢 (開始計時)
    
    alt 權威 DNS 正常回應 (<1.8秒)
        AuthDNS-->>Unbound: 正常回應
        Unbound->>Cache: 更新快取
        Unbound-->>Client: 返回最新數據
    else 權威 DNS 超時或故障 (>1.8秒)
        Note over Unbound: 觸發 serve-expired 機制
        
        Unbound->>Cache: 檢查過期快取
        Cache-->>Unbound: 有過期數據 (2小時前)
        
        alt 過期數據 < 24小時
            Note over Unbound: serve-expired-ttl: 86400
            Unbound->>Unbound: 設置回應 TTL = 30秒
            Note over Unbound: serve-expired-reply-ttl: 30
            Unbound-->>Client: 返回過期快取 (TTL=30s)
            Note over Client: 客戶端在 30秒後重試
        else 過期數據 > 24小時
            Unbound-->>Client: 返回 SERVFAIL
        end
    end
```

**DNSTAP 數據處理流程**

```mermaid
sequenceDiagram
    participant DNSdist as dnsdist
    participant Unbound as Unbound
    participant Socket as Unix Socket
    participant Vector as Vector Sidecar
    participant Anon as IP 匿名化
    participant CH as ClickHouse
    
    DNSdist->>Socket: 發送 DNSTAP 消息<br/>(Protocol Buffers)
    Unbound->>Socket: 發送 DNSTAP 消息
    
    Socket->>Vector: 讀取 DNSTAP 消息
    
    Vector->>Vector: 解析 Protocol Buffers
    
    Vector->>Anon: 提取客戶端 IP
    
    Note over Anon: IPv4: 203.0.113.1 → 203.0.113.0<br/>IPv6: 2001:db8::1 → 2001:db8::0
    
    Anon->>Vector: 匿名化後的數據
    
    Vector->>Vector: 添加元數據<br/>(timestamp, namespace, cluster)
    
    Vector->>Vector: 批量緩衝<br/>(10MB 或 5秒)
    
    Vector->>CH: 批量寫入<br/>(gzip 壓縮)
    
    CH->>CH: 存儲到 dnstap_logs 表
```

**健康檢查時序圖**

```mermaid

    participant Client as 客戶端
    participant DNSdist as dnsdist
    participant Unbound as Unbound
    participant Monitor as 健康監控
    
    Note over DNSdist: Lazy Health Check 模式<br/>lazyHealthCheckThreshold: 30%
    
    loop 正常查詢
        Client->>DNSdist: DNS 查詢

    loop 查詢失敗
        Client->>DNSdist: DNS 查詢
        DNSdist->>Unbound: 轉發查詢
        Unbound-->>DNSdist: 超時/錯誤
        DNSdist-->>Client: SERVFAIL
        Note over Monitor: 失敗計數: 35/100 (35%)
    end
    
    Note over Monitor: 失敗率 > 30%<br/>觸發主動健康檢查
    
    Monitor->>Unbound: 主動健康檢查探針
    Unbound-->>Monitor: 無回應
    
    Monitor->>DNSdist: 標記 Unbound 為不健康
    DNSdist->>DNSdist: 從池中移除 Unbound
    
    Note over DNSdist: 所有後端不可用
    
    Client->>DNSdist: DNS 查詢
    DNSdist->>DNSdist: 檢查 Packet Cache
    alt 有快取
        DNSdist-->>Client: 返回快取 (可能過期)
    else 無快取
        DNSdist-->>Client: 返回 SERVFAIL
    end
    
    Note over Unbound: 服務恢復
    
    Monitor->>Unbound: 定期健康檢查
    Unbound-->>Monitor: 正常回應
    
    Monitor->>DNSdist: 標記 Unbound 為健康
    DNSdist->>DNSdist: 重新加入池中
    
    Client->>DNSdist: DNS 查詢
    DNSdist->>Unbound: 轉發查詢
    Unbound-->>DNSdist: 成功回應
    DNSdist-->>Client: 返回結果
```
