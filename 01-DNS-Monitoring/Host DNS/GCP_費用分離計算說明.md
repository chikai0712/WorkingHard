# GCP 費用分離計算說明

## 📋 概述

本文件將 **Google Cloud 自建 DNS Resolver** 的費用和 **DNS 查詢費用（Google Cloud DNS）** 分開計算，方便您清楚了解各項成本。

---

## 💰 費用結構

### 一、自建 DNS Resolver 費用

這是在 GCP 上運行自己的 DNS Resolver（dnsdist + Unbound）的費用。

#### 1. DNS VM（2 台 Active-Active Master）
- **機型**：`n2-standard-8`（8 vCPU / 32GB RAM）
- **數量**：2 台
- **區域**：跨區域部署（如 `asia-east1` 和 `asia-northeast1`）
- **運算費用**：$0.379 / 小時 × 730 小時 × 2 = **$553 / 月**
- **磁碟費用**：(100GB + 200GB) × $0.17/GB-月 × 2 = **$102 / 月**
- **小計**：**$655 / 月**

#### 2. 監控 VM（1 台）
- **機型**：`n2-standard-4`（4 vCPU / 16GB RAM）
- **數量**：1 台
- **運算費用**：$0.1895 / 小時 × 730 小時 = **$138 / 月**
- **磁碟費用**：(300GB + 1000GB) × $0.17/GB-月 = **$221 / 月**
- **小計**：**$359 / 月**

#### 3. Bastion VM（1 台）
- **機型**：`e2-medium`（2 vCPU / 4GB RAM）
- **數量**：1 台
- **運算費用**：$0.046 / 小時 × 730 小時 = **$34 / 月**
- **磁碟費用**：50GB × $0.17/GB-月 = **$9 / 月**
- **小計**：**$43 / 月**

#### 4. Load Balancer（1 個）
- **類型**：Global TCP Load Balancer
- **數量**：1 個
- **費用**：**$20 / 月**

#### 自建 DNS Resolver 總計
- **不含折扣**：約 **$1,077 / 月**
- **含折扣（1 年承諾使用）**：約 **$754 / 月**（省 30%）

---

### 二、DNS 查詢費用（Google Cloud DNS）

這是使用 Google Cloud DNS 作為權威 DNS 服務的查詢費用。

#### 1. 查詢費用（Query）
- **查詢量**：20k QPS = 52 億次/月 = 52,000 百萬次/月
- **價格結構**：
  - 前 10 億次/月：$0.40 / 百萬次
  - 超過 10 億次/月：$0.20 / 百萬次
- **計算**：
  - 前 10 億次：10,000 百萬次 × $0.40 = **$4,000**
  - 剩餘 42 億次：42,000 百萬次 × $0.20 = **$8,400**
  - **小計**：**$12,400 / 月**

#### 2. 託管區域費用（Managed Zone）
- **數量**：假設 1 個託管區域
- **價格**：前 25 個託管區域 $0.20 / 月
- **費用**：**$0.20 / 月**

#### 3. 健康檢查費用（Health Checks）
- **數量**：假設 2 個快速健康檢查
- **價格**：$0.50 / 個/月
- **費用**：2 × $0.50 = **$1.00 / 月**

#### DNS 查詢費用總計
- **總計**：約 **$12,401.20 / 月**
- **說明**：查詢費用佔總費用的 99.98% 以上

---

### 三、網路流量費用（自建 DNS Resolver 出站流量）

這是自建 DNS Resolver 對外提供服務的網路流量費用。

#### 1. DNS 出站流量
- **流量**：20k QPS × 80 bytes × 86,400 秒 × 30 天 = **4,147 GB/月**
- **價格結構**（GCP 階梯價格）：
  - 前 1 TB/月：$0.12 / GB
  - 1-10 TB/月：$0.11 / GB
  - 10-100 TB/月：$0.08 / GB
- **計算**：
  - 前 1 TB：1,024 GB × $0.12 = **$122.88**
  - 剩餘 3,123 GB：3,123 GB × $0.11 = **$343.53**
  - **小計**：**$466 / 月**

#### 2. 監控流量
- **流量**：Prometheus + Loki 約 **70 GB/月**
- **計算**：70 GB × $0.12 = **$8.40 / 月**

#### 網路流量費用總計
- **總計**：約 **$474 / 月**

## 🔍 關鍵發現

### 1. DNS 查詢費用是主要成本
- **佔總費用 90% 以上**
- 自建 DNS Resolver 費用相對較小（約 5-8%）
- 網路流量費用也相對較小（約 3-4%）

### 2. 自建 DNS Resolver 成本合理
- **不含折扣**：約 $1,077 / 月
- **含折扣**：約 $754 / 月
- 相比 DNS 查詢費用，自建成本相對較低

### 3. 成本優化建議

#### 減少 DNS 查詢費用
- ✅ **使用 DNS 快取**：減少對權威 DNS 的查詢
- ✅ **提高 TTL**：減少查詢頻率
- ✅ **使用 CDN**：減少 DNS 查詢需求
- ✅ **優化查詢模式**：避免不必要的查詢

#### 優化自建 DNS Resolver
- ✅ **使用承諾使用折扣**：可省約 30% 運算費用
- ✅ **優化磁碟配置**：根據實際需求調整磁碟大小
- ✅ **監控流量優化**：減少監控數據傳輸

#### 優化網路流量
- ✅ **提高快取命中率**：減少出站流量
- ✅ **使用 CDN**：降低流量費用

## 💡 使用試算表

### 步驟
1. 在 Google 試算表選取 A1
2. 貼上 `GCP_費用分離計算.txt` 的內容
3. 試算表會自動計算：
   - 自建 DNS Resolver 費用
   - DNS 查詢費用
   - 網路流量費用
   - 總計（含/不含折扣）
4. 可調整「QPS」參數來更新所有費用

### 參數說明
- **QPS**：每秒查詢數（預設 20,000）
- **DNS 查詢次數**：自動計算（QPS × 86,400 × 30 / 1,000,000）
- **DNS 出站流量**：自動計算（QPS × 80 bytes × 86,400 × 30 / 1024³）

---

## 📝 注意事項

1. **DNS 查詢費用無折扣**：Google Cloud DNS 查詢費用不適用承諾使用折扣
2. **流量費用無折扣**：網路流量費用也不適用承諾使用折扣
3. **自建費用可折扣**：運算和磁碟費用適用承諾使用折扣（約 30%）
4. **實際費用**：建議使用 GCP 定價計算器進行精確估算
