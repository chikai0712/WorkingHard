# AWS / GCP 費用估算說明（中型 20k QPS）

## 一、規格對應

### DNS VM（3 台）⭐ 推薦
- **AWS**：`m6i.2xlarge`（8 vCPU / 32GB RAM）
- **GCP**：`n2-standard-8`（8 vCPU / 32GB RAM）
- **磁碟**：100GB SSD（系統碟）+ 200GB HDD（日誌碟）
- **部署**：跨 3 個可用區（高可用性）

### 監控 VM（1 台）
- **AWS**：`m6i.xlarge`（4 vCPU / 16GB RAM）
- **GCP**：`n2-standard-4`（4 vCPU / 16GB RAM）
- **磁碟**：300GB SSD（系統碟）+ 1000GB HDD（日誌碟）

### Bastion VM（1 台）
- **AWS**：`t3.medium`（2 vCPU / 4GB RAM）
- **GCP**：`e2-medium`（2 vCPU / 4GB RAM）
- **磁碟**：50GB SSD（系統碟）

## 三、費用計算公式

### 運算費用
```
月費 = 單價/小時 × 730 小時/月 × 數量
```

### 磁碟費用
```
月費 = (系統碟 GB + 日誌碟 GB) × 磁碟單價/GB-月 × 數量
```

### 網路流量費用
```
出站流量 (GB/月) = QPS × 回應封包大小 (80 bytes) × 86400 秒 × 30 天 / 1024³
AWS 費用 = 出站流量 × 階梯價格（前 10TB: $0.09/GB）
GCP 費用 = 出站流量 × 階梯價格（前 1TB: $0.12/GB, 1-10TB: $0.11/GB）
```

### 總費用
```
總月費 = 運算月費 + 磁碟月費 + 網路流量費用
```

## 四、估算結果（不含折扣）

### AWS 總月費
- **DNS VM（3台）**：$0.384 × 730 × 3 = **$841.00**
- **監控 VM（1台）**：$0.192 × 730 × 1 = **$140.16**
- **Bastion VM（1台）**：$0.0416 × 730 × 1 = **$30.37**
- **磁碟費用**：(100+200)×0.08×3 + (300+1000)×0.08×1 + 50×0.08×1 = **$108.00**
- **網路流量**：4,147 GB × $0.09 = **$373.00**（20k QPS 出站流量）
- **總計**：約 **$1,492 / 月**

### GCP 總月費
- **DNS VM（3台）**：$0.379 × 730 × 3 = **$830.00**
- **監控 VM（1台）**：$0.1895 × 730 × 1 = **$138.24**
- **Bastion VM（1台）**：$0.046 × 730 × 1 = **$33.58**
- **磁碟費用**：(100+200)×0.17×3 + (300+1000)×0.17×1 + 50×0.17×1 = **$230.00**
- **網路流量**：4,147 GB（前 1TB $0.12/GB，剩餘 $0.11/GB）= **$466.00**（20k QPS 出站流量）
- **總計**：約 **$1,698 / 月**

## 六、網路流量詳細說明

### 流量計算（20k QPS）
- **DNS 回應封包**：平均 80 bytes
- **出站流量**：20,000 QPS × 80 bytes × 86,400 秒 × 30 天 = **4,147 GB/月**
- **監控流量**：約 70 GB/月（Prometheus + Loki）
- **總出站流量**：約 **4,147 GB/月**

### AWS 流量價格（ap-northeast-1）
- **前 10 TB/月**：$0.09 / GB
- **10-40 TB/月**：$0.085 / GB
- **40-100 TB/月**：$0.07 / GB
- **20k QPS 費用**：4,147 GB × $0.09 = **$373 / 月**

### GCP 流量價格（asia-northeast1）
- **前 1 TB/月**：$0.12 / GB
- **1-10 TB/月**：$0.11 / GB
- **10-100 TB/月**：$0.08 / GB
- **20k QPS 費用**：
  - 前 1 TB：1,024 GB × $0.12 = $122.88
  - 剩餘 3,123 GB：3,123 GB × $0.11 = $343.53
  - **總計**：**$466 / 月**

### 流量優化建議
- **提高快取命中率**：從 50% 提升至 80% 可減少 60% 流量費用
- **使用 CDN**：可降低 20-30% 流量費用
- **監控實際流量**：設定流量告警，避免超支

## 七、GCP 單一平台 Active-Active 方案（推薦）⭐

### 方案概述
選擇 **GCP 單一平台建置兩台 Active-Active Master 主機**，這是一個非常務實且高效的決策！

### 核心優勢

#### 1. 極速內網 (VPC Network)
- **全球光纖骨幹**：兩台主機雖在不同區域（如台灣與東京），但走的是 Google 全球光纖骨幹
- **低延遲同步**：VPC 內網延遲通常 < 5ms，即使跨區域
- **高頻寬**：VPC 內網頻寬不受限制，可達到 10 Gbps 以上
- **實際效益**：DNS 同步速度極快，監控數據傳輸無壓力

#### 2. 管理單純化
- **統一帳單**：所有資源在同一帳單，費用管理簡單
- **統一 IAM 權限**：一套權限管理系統，無需跨平台配置
- **統一監控介面**：Cloud Operations 統一監控
- **實際效益**：降低管理成本，簡化權限管理

#### 3. 異地備援 (Disaster Recovery)
- **物理級別備援**：不同區域 = 不同數據中心 = 不同物理位置
- **自動故障轉移**：GCP Load Balancer 可自動偵測故障並轉移流量
- **實際效益**：符合災難備援要求，零停機時間

## 九、使用試算表

### 主要費用試算表
1. 在 Google 試算表選取 A1
2. 貼上 `雲端費用估算_AWS_GCP.txt` 的內容
3. 試算表會自動計算運算、磁碟、流量費用
4. 可調整「單價/小時」、「磁碟單價」、「QPS」欄位來更新價格

### 流量估算試算表
1. 在 Google 試算表選取 A1
2. 貼上 `網路流量估算試算表.txt` 的內容
3. 可輸入不同 QPS 值來計算對應的流量和費用
4. 查看 `網路流量估算說明.md` 了解詳細計算公式
