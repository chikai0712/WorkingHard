# 快速開始指南

## 5 分鐘快速設定

### 1️⃣ 取得服務帳號金鑰

**完整指南：** 請查看 [取得服務金鑰指南.md](取得服務金鑰指南.md)

**快速連結：**
- [Google Cloud Console](https://console.cloud.google.com/)
- [建立服務帳號](https://console.cloud.google.com/iam-admin/serviceaccounts)
- [啟用 Cloud DNS API](https://console.cloud.google.com/apis/library/dns.googleapis.com)

### 2️⃣ 下載金鑰檔案

1. 建立服務帳號（名稱：`dns-manager`）
2. 授予角色：**Cloud DNS Admin**
3. 建立金鑰：選擇 **JSON** 格式
4. 下載檔案並重新命名為：`service-account-key.json`
5. 將檔案放在專案目錄中

### 3️⃣ 更新設定

編輯 `example.py`：

```python
PROJECT_ID = "your-project-id"           # ← 你的 GCP 專案 ID
SERVICE_ACCOUNT_KEY = "service-account-key.json"  # ← 金鑰檔案名稱
ZONE_NAME = "my-dns-zone"                # ← 你的 Zone 名稱
```

### 4️⃣ 執行測試

```bash
cd "/Users/ckchiu/Desktop/Project/DNS/google API"
source venv/bin/activate
python example.py
```
