## 🎯 使用建議

### 檢測策略

1. **第一步：國際 DNS 檢測**
   - 使用 20 個國際 DNS
   - 成功率 < 50% → 域名無效
   - 成功率 >= 50% → 繼續下一步

2. **第二步：中國 ISP 檢測**
   - 使用 42 個中國 ISP DNS
   - 國際 OK + 中國失敗 → 中國封鎖
   - 可分析具體是哪個運營商封鎖

3. **第三步：亞太地區檢測**（可選）
   - 使用 16 個亞太 DNS
   - 了解周邊國家可訪問性

# 不需要每次都檢測所有 DNS
