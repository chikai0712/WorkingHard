# Port Forward 已啟動 ✅

## 當前狀態

- ✅ Port forward 運行中（UDP PID: 66048, TCP PID: 66049）
- ✅ 端口 53 已釋放
- ✅ DNS 服務器運行中

## 下一步操作

### 步驟 1: 設定 macOS DNS（重要！）

1. 開啟「系統設定」
2. 點選「網路」>「Wi-Fi」>「詳細資訊」
3. 點選「DNS」標籤
4. **移除所有現有的 DNS 伺服器**（點選「-」按鈕）
5. 點選「+」新增：`127.0.0.1`
6. 按「好」→「套用」

> ⚠️ **注意**：這會讓你的 Mac 使用測試 DNS，測試完成後記得恢復！

### 步驟 2: 清空 DNS Cache 並測試

在終端機執行：

```bash
sudo killall -HUP mDNSResponder
dig app.example.com
```

如果看到回應（即使有警告訊息），表示設定成功！

### 步驟 3: 開始抓包測試（量測 retry 行為）

#### 3.1 獲取你的 Mac IP

```bash
MY_IP=$(ipconfig getifaddr en0)
echo "你的 Mac IP: $MY_IP"
```

#### 3.2 在另一個終端機視窗開始抓包

```bash
sudo tcpdump -i any udp port 53 and host $MY_IP -w ~/dns_retry_test.pcap
```

#### 3.3 在第三個終端機觸發查詢

```bash
# 清空 cache
sudo killall -HUP mDNSResponder

# 觸發查詢（這會觸發 macOS 的 retry 機制）
dig app.example.com
```

#### 3.4 停止抓包

在抓包的終端機按 `Ctrl+C` 停止 tcpdump。

#### 3.5 分析結果

```bash
# 用 Wireshark 打開（如果有安裝）
open ~/dns_retry_test.pcap

# 或在終端機查看
tcpdump -r ~/dns_retry_test.pcap -n -ttt
```

### 步驟 4: 測試完成後恢復

```bash
cd /Users/ckchiu/Desktop/Project/DNS-HA-Simulator

# 停止 port forward
sudo ./scripts/setup-dns-forward.sh stop

# 恢復 macOS DNS 服務
sudo launchctl load -w /System/Library/LaunchDaemons/com.apple.mDNSResponder.plist

# 恢復 macOS DNS 設定（在系統設定中改回原本的 DNS 伺服器）
```
