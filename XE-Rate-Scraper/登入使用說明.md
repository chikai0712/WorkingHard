# XE.com 登入使用說明

## 🔐 登入方式

本工具支援多種方式提供登入憑證：

### 方法 1: 使用環境變數（推薦，最安全）

1. **創建 `.env` 檔案**（從 `.env.example` 複製）：
   ```bash
   cp .env.example .env
   ```

2. **編輯 `.env` 檔案**，填入您的帳號資訊：
   ```
   XE_EMAIL=your-email@example.com
   XE_PASSWORD=your-password
   ```

3. **執行抓取**：
   ```bash
   ./run.sh
   # 或
   python3 xe_scraper.py
   ```

**優點**：
- ✅ 密碼不會出現在命令列歷史記錄中
- ✅ `.env` 檔案已在 `.gitignore` 中，不會被提交
- ✅ 方便管理多個帳號

### 方法 2: 使用命令列參數

```bash
python3 xe_scraper.py --email your-email@example.com --password your-password
```

**注意**：密碼會出現在命令列歷史記錄中，不建議在共享環境使用。

### 方法 3: 交互式輸入（僅在非 headless 模式下）

如果只提供了 Email，程式會提示輸入密碼：

```bash
python3 xe_scraper.py --email your-email@example.com --no-headless
# 然後會提示輸入密碼
```

### 方法 4: 混合使用

```bash
# Email 從環境變數，密碼從命令列
export XE_EMAIL=your-email@example.com
python3 xe_scraper.py --password your-password
```

## 📋 完整範例

### 使用環境變數登入

```bash
# 1. 設置環境變數
export XE_EMAIL=your-email@example.com
export XE_PASSWORD=your-password

# 2. 執行抓取
python3 xe_scraper.py
```

### 使用 .env 檔案

```bash
# 1. 創建 .env 檔案
cat > .env <<EOF
XE_EMAIL=your-email@example.com
XE_PASSWORD=your-password
EOF

# 2. 執行抓取（會自動讀取 .env）
python3 xe_scraper.py
```

### 使用命令列參數

```bash
python3 xe_scraper.py \
  --email your-email@example.com \
  --password your-password \
  --no-headless
```

## 🔒 安全建議

1. **不要將 `.env` 檔案提交到版本控制**
   - `.env` 已在 `.gitignore` 中
   - 只提交 `.env.example` 作為範本

2. **使用環境變數而非命令列參數**
   - 命令列參數會出現在程序列表中
   - 環境變數更安全

3. **定期更換密碼**
   - 如果懷疑憑證洩露，立即更換

---

## ⚠️ 注意事項

1. **登入需要 Selenium**
   - 如果使用 `--no-selenium`，登入功能將無法使用
   - 登入功能僅在 Selenium 模式下可用

2. **登入可能失敗的原因**
   - 帳號密碼錯誤
   - 網站結構改變（選擇器需要更新）
   - 需要驗證碼（目前不支援）
   - 帳號被鎖定

3. **如果登入失敗**
   - 使用 `--no-headless` 查看瀏覽器行為
   - 檢查帳號密碼是否正確
   - 查看控制台輸出的錯誤訊息

## 🧪 測試登入

### 測試登入功能（不抓取數據）

```bash
python3 -c "
from xe_scraper import XERateScraper
import os

email = os.getenv('XE_EMAIL') or input('Email: ')
password = os.getenv('XE_PASSWORD') or input('Password: ')

with XERateScraper(headless=False, email=email, password=password) as scraper:
    if scraper.login():
        print('✅ 登入成功！')
        input('按 Enter 關閉瀏覽器...')
    else:
        print('❌ 登入失敗')
"
```

## 📝 故障排除

### 問題：找不到登入表單元素

**解決方案**：
1. 使用 `--no-headless` 查看實際頁面
2. 網站結構可能已更改，需要更新選擇器
3. 檢查是否需要等待更長時間讓 JavaScript 載入

### 問題：登入後仍被重定向到登入頁面

**解決方案**：
1. 檢查帳號密碼是否正確
2. 檢查是否需要驗證碼（目前不支援）
3. 檢查帳號是否被鎖定

### 問題：環境變數未讀取

**解決方案**：
```bash
# 確保已安裝 python-dotenv
pip install python-dotenv

# 或手動設置環境變數
export XE_EMAIL=your-email@example.com
export XE_PASSWORD=your-password
```
