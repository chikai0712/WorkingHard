### 步驟 1: 啟動服務

```bash
# 啟動後端
cd Stock_Analize/backend
source venv/bin/activate
python main.py

# 啟動前端（另一個終端）
cd Stock_Analize/frontend
npm run dev
```

### 步驟 2: 檢查資料庫

```bash
cd Stock_Analize/backend
source venv/bin/activate
python -c "
from database.session import SessionLocal
from database.models_futures import MarginTradingData
db = SessionLocal()
count = db.query(MarginTradingData).count()
print(f'資料筆數: {count}')
if count > 0:
    latest = db.query(MarginTradingData).order_by(MarginTradingData.date.desc()).first()
    print(f'最新資料: {latest.date}')
db.close()
"
```
