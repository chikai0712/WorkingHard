### 3. ✅ 資料保存30天，自動清理

**功能**:
- 自動清理超過30天的舊資料
- 每天 02:00 執行清理任務

**實現**:
```python
def cleanup_old_data():
    """清理超過30天的舊資料"""
    cutoff_date = date.today() - timedelta(days=30)
    deleted_count = db.query(MarginTradingData).filter(
        MarginTradingData.date < cutoff_date
    ).delete()
```

**狀態**: ✅ 已完成

## 使用方式

自動排程會在後端服務啟動時自動啟用：

