# 快速開始指南

## 📋 前置需求

- **Python 3.10+**
- **Node.js 16+**
- **npm** 或 **yarn**

## 🚀 安裝步驟

### 1. 後端設置

#### 1.1 進入後端目錄
```bash
cd Stock_Analize/backend
```

#### 1.2 建立虛擬環境（建議）
```bash
# macOS/Linux
python3 -m venv venv
source venv/bin/activate

# Windows
python -m venv venv
venv\Scripts\activate
```

#### 1.3 安裝依賴
```bash
pip install -r requirements.txt
```

#### 1.4 設定環境變數
```bash
cp .env.example .env
```

編輯 `.env` 檔案，確認以下設定：
```env
HOST=0.0.0.0
PORT=8000
DATABASE_URL=sqlite:///./stock_data.db
DEBUG=True
```

#### 1.5 啟動後端服務
```bash
python main.py
```

或者使用 uvicorn 直接啟動：
```bash
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

**後端服務將在**：`http://localhost:8000`

**API 文件**：`http://localhost:8000/docs`（Swagger UI）

---

### 2. 前端設置

#### 2.1 進入前端目錄
```bash
cd Stock_Analize/frontend
```

#### 2.2 安裝依賴
```bash
npm install
```

#### 2.3 設定環境變數
```bash
cp .env.example .env
```

確認 `.env` 檔案中的 API 基礎 URL：
```env
VITE_API_BASE_URL=http://localhost:8000/api
```

#### 2.4 啟動前端開發伺服器
```bash
npm run dev
```

**前端服務將在**：`http://localhost:5173`

## 📊 初始資料設置

### 新增監控股票

#### 方法 1：使用 API

```bash
# 新增台積電（台股）
curl -X POST "http://localhost:8000/api/stocks" \
  -H "Content-Type: application/json" \
  -d '{
    "symbol": "2330",
    "market": "TW",
    "name": "台積電"
  }'

# 新增蘋果（美股）
curl -X POST "http://localhost:8000/api/stocks" \
  -H "Content-Type: application/json" \
  -d '{
    "symbol": "AAPL",
    "market": "US",
    "name": "Apple Inc."
  }'
```

#### 方法 2：使用 Python 腳本

建立 `backend/scripts/add_stocks.py`：

```python
from database.session import SessionLocal
from services.stock_service import StockService

db = SessionLocal()
try:
    stock_service = StockService(db)
    
    # 新增台股
    stocks_tw = [
        {"symbol": "2330", "market": "TW", "name": "台積電"},
        {"symbol": "2317", "market": "TW", "name": "鴻海"},
        {"symbol": "2454", "market": "TW", "name": "聯發科"},
    ]
    
    for stock_data in stocks_tw:
        try:
            stock_service.create_stock(**stock_data)
            print(f"✅ 已新增：{stock_data['symbol']} - {stock_data['name']}")
        except Exception as e:
            print(f"❌ 新增失敗 {stock_data['symbol']}: {e}")
    
finally:
    db.close()
```

執行：
```bash
cd backend
python scripts/add_stocks.py
```

### 觸發資料抓取

#### 方法 1：等待定時任務（自動）
系統會根據 `.env` 中的 `UPDATE_INTERVAL_MINUTES` 設定自動抓取資料。

#### 方法 2：手動觸發（測試用）

建立 `backend/scripts/manual_update.py`：

```python
from database.session import SessionLocal
from crawler.stock_crawler import StockCrawler

db = SessionLocal()
try:
    crawler = StockCrawler(db)
    results = crawler.update_all_stocks()
    print(f"更新完成：成功 {results['success']}，失敗 {results['failed']}")
finally:
    db.close()
```

執行：
```bash
cd backend
python scripts/manual_update.py
```

## 🔧 常見問題

### 問題 1：後端無法啟動

**錯誤**：`ModuleNotFoundError: No module named 'fastapi'`

**解決**：
- 確認已啟動虛擬環境
- 重新安裝依賴：`pip install -r requirements.txt`

### 問題 2：前端無法連接後端

**錯誤**：API 請求失敗或 CORS 錯誤

**解決**：
1. 確認後端服務正在運行（訪問 `http://localhost:8000/health`）
2. 檢查 `frontend/.env` 中的 `VITE_API_BASE_URL` 是否正確
3. 確認 `backend/.env` 中的 `CORS_ORIGINS` 包含前端 URL

### 問題 3：資料抓取失敗

**錯誤**：無法取得股票資料

**解決**：
1. 檢查網路連線
2. 確認股票代號格式正確（台股：2330，美股：AAPL）
3. 檢查 yfinance 是否正常工作：`python -c "import yfinance as yf; print(yf.Ticker('2330.TW').info)"`

### 問題 4：資料庫錯誤

**錯誤**：資料庫表不存在

**解決**：
```bash
cd backend
python -c "from database.session import init_db; init_db()"
```

或者重新啟動後端服務，系統會自動初始化資料庫。
