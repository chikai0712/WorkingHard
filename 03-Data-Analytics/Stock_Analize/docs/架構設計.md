## 🔄 資料流程

### 1. 資料抓取流程

```
定時任務觸發
    ↓
選擇股票代號列表
    ↓
並發請求多個資料來源
    ↓
資料驗證與清理
    ↓
存入資料庫
    ↓
觸發 WebSocket 推送（可選）
```

### 2. 前端資料請求流程

```
使用者開啟儀表板
    ↓
前端發起 HTTP 請求
    ↓
後端 API 處理請求
    ↓
從資料庫/快取查詢資料
    ↓
返回 JSON 格式資料
    ↓
前端渲染圖表與表格
    ↓
定時輪詢更新（或 WebSocket）
```

## 🗄️ 資料庫設計

### 股票基本資料表 (stocks)

| 欄位 | 類型 | 說明 |
|------|------|------|
| id | INTEGER PRIMARY KEY | 主鍵 |
| symbol | VARCHAR(20) | 股票代號 |
| market | VARCHAR(10) | 市場（TW/US/...） |
| name | VARCHAR(100) | 股票名稱 |
| industry | VARCHAR(50) | 產業分類 |
| created_at | DATETIME | 建立時間 |
| updated_at | DATETIME | 更新時間 |

### 股票價格資料表 (stock_prices)

| 欄位 | 類型 | 說明 |
|------|------|------|
| id | INTEGER PRIMARY KEY | 主鍵 |
| stock_id | INTEGER | 關聯 stocks.id |
| date | DATE | 交易日期 |
| open | DECIMAL(10,2) | 開盤價 |
| high | DECIMAL(10,2) | 最高價 |
| low | DECIMAL(10,2) | 最低價 |
| close | DECIMAL(10,2) | 收盤價 |
| volume | BIGINT | 成交量 |
| created_at | DATETIME | 建立時間 |

### 技術指標表 (technical_indicators)

| 欄位 | 類型 | 說明 |
|------|------|------|
| id | INTEGER PRIMARY KEY | 主鍵 |
| stock_id | INTEGER | 關聯 stocks.id |
| date | DATE | 計算日期 |
| indicator_type | VARCHAR(20) | 指標類型（MA/RSI/MACD） |
| value | DECIMAL(10,4) | 指標值 |
| period | INTEGER | 計算週期 |

## 🔌 API 端點設計

### 股票列表相關

- `GET /api/stocks` - 取得所有監控股票列表
- `GET /api/stocks/{symbol}` - 取得特定股票資訊
- `POST /api/stocks` - 新增監控股票
- `DELETE /api/stocks/{symbol}` - 移除監控股票

### 價格資料相關

- `GET /api/stocks/{symbol}/price` - 取得最新價格
- `GET /api/stocks/{symbol}/history` - 取得歷史價格資料
- `GET /api/stocks/{symbol}/history?start_date=2024-01-01&end_date=2024-12-31` - 取得指定日期區間資料

### 技術指標相關

- `GET /api/stocks/{symbol}/indicators` - 取得技術指標
- `GET /api/stocks/{symbol}/indicators/{indicator_type}` - 取得特定指標

### 統計資訊

- `GET /api/dashboard/summary` - 取得儀表板摘要統計
- `GET /api/dashboard/top-gainers` - 取得漲幅排行榜
- `GET /api/dashboard/top-losers` - 取得跌幅排行榜

## 🔄 定時任務設計

### 任務類型

1. **即時價格更新任務**
   - 執行頻率：每 5 分鐘（交易時間）
   - 功能：抓取最新價格資料
   - 觸發時間：09:00-13:30（台股）

2. **歷史資料補齊任務**
   - 執行頻率：每天一次
   - 功能：補齊缺失的歷史資料
   - 觸發時間：收盤後（15:00）

3. **技術指標計算任務**
   - 執行頻率：每天收盤後
   - 功能：計算所有技術指標
   - 觸發時間：15:30

4. **資料清理任務**
   - 執行頻率：每週一次
   - 功能：清理舊資料、備份
   - 觸發時間：週日 02:00

## 🎨 前端組件設計

### 主要頁面組件

1. **Dashboard.vue** - 主儀表板
   - 顯示所有監控股票的摘要
   - 快速查看漲跌幅
   - 熱門股票排行

2. **StockDetail.vue** - 股票詳情頁
   - 詳細價格資訊
   - K 線圖表
   - 技術指標圖表
   - 相關新聞

3. **StockList.vue** - 股票列表
   - 可排序的股票表格
   - 搜尋與篩選功能
   - 批量操作

### 可重用組件

1. **PriceChart.vue** - 價格圖表組件
   - 支援 K 線圖、折線圖
   - 可切換時間週期

2. **StockCard.vue** - 股票卡片組件
   - 顯示基本資訊
   - 價格變動顏色標示

3. **IndicatorPanel.vue** - 技術指標面板
   - 顯示多種技術指標
   - 可切換指標類型
