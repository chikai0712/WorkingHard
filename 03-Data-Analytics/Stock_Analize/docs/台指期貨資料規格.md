# 台指期貨資料規格文件

根據需求圖片，本系統需要抓取以下台指期貨相關資料：

## 📊 資料項目清單

### 1. 三大法人買賣超 (Institutional Trading)

**資料來源**: 台灣期貨交易所 (TAIFEX)

**顯示欄位**:
- 日期
- 加權指數 / 櫃買指數
- 外資：買進、賣出、現貨差額(億)、昨日差異
- 投信：買進、賣出、現貨差額(億)、昨日差異
- 自營商：買進、賣出、現貨差額(億)、昨日差異
  - 自營自買：買進、賣出、現貨差額(億)
  - 自營避險：買進、賣出、現貨差額(億)
- 三大法人合計：現貨差額(億)

**API 端點**: `GET /api/futures/institutional`

---

### 2. 融資融券餘額 (Margin Trading)

**資料來源**: 台灣證券交易所 (TWSE) / 櫃買中心 (TPEX)

**顯示欄位**:
- 日期
- 加權指數 / 櫃買指數
- 融資餘額(億)、昨日差異
- 融券餘額(張)、昨日差異
- 借券賣出(張)、昨日差異
- 相關指數價格、漲跌幅

**API 端點**: `GET /api/futures/margin-trading`

---

### 3. 台指期貨未平倉量 (Futures Open Interest)

**資料來源**: 台灣期貨交易所 (TAIFEX)

**顯示欄位**:
- 日期
- 外資：未平倉口數、昨日差異
- 投信：未平倉口數、昨日差異
- 自營：未平倉口數、昨日差異
- 總計：未平倉口數

**API 端點**: `GET /api/futures/open-interest`

---

### 4. 台指選擇權未平倉量 (Options Open Interest)

**資料來源**: 台灣期貨交易所 (TAIFEX)

**顯示欄位**:
- 日期
- 外資：未平倉口數、昨日差異
- 投信：未平倉口數、昨日差異
- 自營：未平倉口數、昨日差異
- P/C 比：成交量P/C比、未平倉P/C比

**API 端點**: `GET /api/futures/options/open-interest`

---

### 5. 期貨法人&散戶淨未平倉

**資料來源**: 台灣期貨交易所 (TAIFEX)

**顯示欄位**:
- 日期
- 外資：淨未平倉、淨未平倉增減
- 10大交易人：淨未平倉、淨未平倉增減
- 10大特定人：淨未平倉、淨未平倉增減
- 散戶：淨未平倉、淨未平倉增減

**API 端點**: `GET /api/futures/open-interest` (已包含在未平倉量 API)

---

### 6. 台指周選擇權每日交易行情

**資料來源**: 台灣期貨交易所 (TAIFEX)

**顯示內容**:
- 日期、合約代號（如：202512F3）
- 各履約價的買權OI、賣權OI
- 買權OI和賣權OI的增減變化
- 履約價範圍：約 26800-28400（以50或100為間隔）

**API 端點**: `GET /api/futures/options/strike-data`

---

### 7. 台指月選擇權每日交易行情

**資料來源**: 台灣期貨交易所 (TAIFEX)

**顯示內容**: 與周選擇權相同，但為月合約

**API 端點**: `GET /api/futures/options/strike-data?period=monthly`

---

### 8. 台指期貨交易及未平倉口數（歷史資料）

**資料來源**: 台灣期貨交易所 (TAIFEX)

**顯示欄位**:
- 日期
- 台股大盤：指數、漲跌、漲跌幅
- 交易口數淨額：外資、投信、自營
- 未平倉口數：外資、投信、自營（含昨日差異）

**API 端點**: `GET /api/futures/daily`

---

### 9. 台指選擇權交易及未平倉口數（歷史資料）

**資料來源**: 台灣期貨交易所 (TAIFEX)

**顯示欄位**:
- 日期
- 台股大盤：指數、漲跌、漲跌幅
- 交易口數淨額：外資、投信、自營
- P/C比：成交量P/C比
- 未平倉口數：外資、投信、自營（含昨日差異）、P/C比

**API 端點**: `GET /api/futures/options/daily`

---

## 🔄 資料更新頻率

- **即時資料**: 盤中每 5 分鐘更新一次
- **盤後資料**: 收盤後（15:30）立即更新
- **歷史資料**: 每日收盤後補齊

---

## 📡 資料來源 API

### 台灣期貨交易所 (TAIFEX)

1. **期貨每日行情**: `https://www.taifex.com.tw/cht/3/futDataDown`
   - down_type=1 (期貨)
   - commodity_id=TX (台指期貨)

2. **三大法人買賣超**: `https://www.taifex.com.tw/cht/3/futDataDown`
   - down_type=2 (三大法人)

3. **選擇權每日行情**: `https://www.taifex.com.tw/cht/3/optDataDown`
   - down_type=1 (選擇權)

### 台灣證券交易所 (TWSE)

1. **融資融券餘額**: `https://www.twse.com.tw/rwd/zh/marginTrading/MI_MARGN`
   - 格式: JSON
   - 參數: date=YYYYMMDD

## 📋 資料庫表結構

詳細資料庫結構請參考：`backend/database/models_futures.py`

主要資料表：
- `futures_contracts` - 期貨合約基本資料
- `futures_daily_data` - 期貨每日交易資料
- `institutional_trading_data` - 三大法人買賣超
- `futures_open_interest` - 期貨未平倉量
- `options_daily_data` - 選擇權每日交易資料
- `options_strike_data` - 選擇權履約價資料
- `margin_trading_data` - 融資融券餘額

---

## 🚀 API 使用範例

### 取得期貨每日資料

```bash
GET /api/futures/daily?symbol=TX&limit=10
```

### 取得三大法人買賣超

```bash
GET /api/futures/institutional?market_type=weighted&limit=30
```

### 取得未平倉量

```bash
GET /api/futures/open-interest?symbol=TX&limit=30
```

### 取得選擇權履約價分布

```bash
GET /api/futures/options/strike-data?period=weekly&date=2025-12-18
```
