### 3. 檢查顯示內容

在期貨儀表板頁面應該看到：

#### ✅ 三大法人買賣超卡片
- 顯示最新日期的三大法人資料
- 表格包含：外資、投信、自營商的買進、賣出、淨買賣超
- 注意：目前 TAIFEX 只解析到自營商資料，外資和投信可能顯示為 0

#### ✅ 三大法人歷史表格（7天）
- 顯示最近 7 天的歷史資料
- 包含外資、投信、自營商的淨買賣超

#### ✅ 融資融券餘額卡片
- 顯示最新日期的融資融券資料
- 包含：融資餘額、融券餘額、借券賣出
- 顯示增減變化

#### ✅ 融資融券歷史表格（7天）
- 顯示最近 7 天的歷史資料
- 包含融資餘額、融券餘額及其增減

### 4. 檢查瀏覽器控制台

打開瀏覽器開發者工具（F12），檢查：
- Console 標籤：是否有錯誤訊息
- Network 標籤：API 請求是否成功（狀態碼 200）

## 可能看到的問題

### 問題 1: 外資和投信顯示為 0

**原因**：TAIFEX 解析邏輯目前只解析到自營商資料

**解決方案**：
- 這是正常的，因為 TAIFEX 的表格結構較複雜
- 可以使用 HiStock 作為資料來源（已在實現中）
- 或改進 TAIFEX 解析邏輯

### 問題 2: 某些日期沒有資料

**原因**：可能是非交易日或資料尚未抓取

**解決方案**：
```bash
# 執行資料抓取腳本補齊資料
cd Stock_Analize/backend
source venv/bin/activate
python scripts/fetch_recent_data.py
```

### 問題 3: CORS 錯誤

**原因**：前端和後端的跨域設定問題

**檢查**：
- 確認 `backend/main.py` 中的 CORS 設定包含前端 URL
- 確認前端 API base URL 設定正確
