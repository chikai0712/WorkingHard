## 如果前端仍無法打開

### 檢查前端服務
```bash
# 檢查前端是否在運行
curl http://localhost:5173

# 如果未運行，啟動前端
cd Stock_Analize/frontend
npm run dev
```

### 檢查後端服務
```bash
# 檢查後端是否在運行
curl http://localhost:8000/health

# 如果未運行，啟動後端
cd Stock_Analize/backend
source venv/bin/activate
python main.py
```
