# 即時買賣資訊來源說明

## 📋 概述

本文檔說明可以取得台股即時交易資訊（買賣價格、成交量等）的各種資料來源。

---

## 🔴 官方來源（免費，但通常有延遲）

### 1. 台灣證券交易所 (TWSE) 公開資料

**網址**: https://www.twse.com.tw/

**提供資料**:
- ✅ 收盤後的完整交易資料（每日）
- ✅ 盤中撮合資訊（有延遲，通常 20 秒）
- ✅ 個股基本資訊
- ✅ 融資融券資料
- ⚠️ **不是真正的即時（通常延遲 20 秒以上）**

**API 端點範例**:
- 個股即時資訊: `https://www.twse.com.tw/rwd/zh/afterTrading/STOCK_DAY`
- 盤中資訊: `https://www.twse.com.tw/rwd/zh/realtimeTrading/STOCK_DAY`

**限制**:
- 有請求頻率限制
- 資料有延遲（非真正的即時）
- 需要解析 HTML 或 JSON 回應

---

### 2. 櫃買中心 (TPEx)

**網址**: https://www.tpex.org.tw/

**提供資料**:
- ✅ 上櫃股票交易資料
- ⚠️ 同樣有延遲問題

---

## 🟡 第三方免費來源（延遲或有限制）

### 1. Yahoo Finance (yfinance)

**當前系統使用**: ✅ 已整合

**Python 套件**: `yfinance`

**特點**:
- ✅ 免費
- ✅ 支援台股（使用 `.TW` 後綴）
- ✅ 容易使用
- ⚠️ **延遲 15-20 分鐘**（不是真正的即時）
- ⚠️ 有速率限制

**使用範例**:
```python
import yfinance as yf

# 台積電
ticker = yf.Ticker("2330.TW")
info = ticker.info
hist = ticker.history(period="1d", interval="1m")  # 最近一天，每分鐘資料
```

**限制**: 
- 延遲 15-20 分鐘
- 不是真正的即時

---

### 2. FinMind API

**網址**: https://finmind.github.io/

**特點**:
- ✅ 提供台灣股票資料
- ✅ 有免費層級
- ⚠️ 免費版本有資料延遲
- ⚠️ 需要註冊 API Key

**API 範例**:
```python
import requests

url = "https://api.finmindtrade.com/api/v4/data"
params = {
    "dataset": "TaiwanStockPrice",
    "data_id": "2330",
    "start_date": "2025-01-01",
    "token": "YOUR_API_KEY"
}
response = requests.get(url, params=params)
```
