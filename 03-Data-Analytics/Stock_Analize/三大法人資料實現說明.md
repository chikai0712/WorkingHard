## TAIFEX API 說明

### API 端點
- URL: `https://www.taifex.com.tw/cht/3/futDataDown`
- 參數: `down_type=2` (三大法人買賣超)

### 回應格式
- 格式: CSV（Big5 編碼）
- 可能需要解析 HTML 頁面（因為 API 可能回傳 HTML 而非 CSV）
