# 借券賣出資料說明

## 問題分析

根據 TWSE API (`/marginTrading/MI_MARGN`) 的回應結構，**這個 API 並不包含借券賣出餘額的資料**。

### API 回應結構

TWSE 融資融券 API 回傳的資料包含：

1. **信用交易統計** (tables[0])
   - 融資(交易單位)
   - 融券(交易單位)
   - 融資金額(仟元)

2. **融資融券彙總** (tables[1])
   - 個股詳細資料

**但沒有借券賣出的資料**

---

## 借券賣出資料來源

借券賣出餘額可能需要從其他 API 取得，常見的有：

1. **證券借貸資訊** (`/lending/t13sa710` 或類似端點)
2. **借券賣出明細表** (`/lending/t13sa710` 或其他)

### 可能的解決方案

#### 方案 1: 使用其他 TWSE API
需要查詢 TWSE 官方文件，找到正確的借券賣出 API 端點。

#### 方案 2: 使用爬蟲
如果 API 不提供，可能需要從網頁爬取資料。

#### 方案 3: 暫時顯示為 0
如果借券賣出資料確實無法取得，可以：
- 在 UI 上標註「資料來源不提供」
- 或隱藏該欄位，直到找到正確的資料來源

---

## 建議

1. **先確認 TWSE 是否有提供借券賣出 API**
   - 查閱 TWSE 官方 API 文件
   - 或聯繫 TWSE 技術支援

2. **如果無法取得**
   - 可以考慮從其他資料來源取得
   - 或暫時在 UI 上標註「資料來源限制」

3. **目前狀態**
   - 借券賣出顯示為 0 是因為資料庫欄位預設值為 0
   - 但實際 API 回應中沒有這個資料
