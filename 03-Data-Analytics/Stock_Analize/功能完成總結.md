## 使用方式

### 自動運行
系統會在後端服務啟動時自動啟用排程，無需手動操作。

```bash
cd backend
source venv/bin/activate
python main.py
```

### 手動觸發（測試用）
```bash
# 手動抓取資料
python scripts/fetch_futures_data.py

# 手動清理舊資料（Python）
from scheduler.futures_tasks import cleanup_old_data
cleanup_old_data()
```

## 資料流程

```
1. 每天 15:30 → 自動抓取當天資料
2. 資料解析 → 提取關鍵資訊
3. 資料儲存 → 存入資料庫
4. 前端顯示 → 顯示最近7天資料
5. 每天 02:00 → 清理超過30天的舊資料
```
