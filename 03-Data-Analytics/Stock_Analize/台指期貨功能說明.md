# 台指期貨資料抓取系統 - 功能說明

## ✅ 已完成的功能

根據您提供的圖片需求，我已經建立了完整的台指期貨資料抓取系統，包含以下功能：

### 1. 資料庫模型 ✅

已建立完整的資料庫模型 (`backend/database/models_futures.py`)：

- ✅ `FuturesContract` - 期貨合約基本資料
- ✅ `FuturesDailyData` - 期貨每日交易資料（價格、成交量、未平倉量）
- ✅ `InstitutionalTradingData` - 三大法人買賣超資料
- ✅ `FuturesOpenInterest` - 期貨未平倉量（包含十大交易人、散戶）
- ✅ `OptionsDailyData` - 選擇權每日交易資料
- ✅ `OptionsStrikeData` - 選擇權履約價分布資料
- ✅ `MarginTradingData` - 融資融券餘額資料

### 2. API 端點 ✅

已建立所有必要的 API 端點：

#### 期貨相關
- ✅ `GET /api/futures/daily` - 期貨每日交易資料
- ✅ `GET /api/futures/institutional` - 三大法人買賣超
- ✅ `GET /api/futures/open-interest` - 期貨未平倉量

#### 選擇權相關
- ✅ `GET /api/futures/options/open-interest` - 選擇權未平倉量
- ✅ `GET /api/futures/options/strike-data` - 選擇權履約價分布（周/月選擇權）
- ✅ `GET /api/futures/options/daily` - 選擇權每日交易資料

#### 融資融券相關
- ✅ `GET /api/futures/margin-trading` - 融資融券餘額（加權指數/櫃買指數）

### 3. 資料抓取模組 ✅

已建立資料抓取架構：

- ✅ `TAIFEXCrawler` - 台灣期貨交易所 API 抓取模組
- ✅ `TAIFEXWebScraper` - 網頁爬蟲備用方案
- ✅ `FuturesScheduler` - 定時任務排程（盤中、盤後更新）

### 4. 定時任務 ✅

已設定期貨資料自動更新：

- ✅ 盤中更新：每 5 分鐘（交易時間）
- ✅ 收盤後更新：每天 15:35
- ✅ 開盤前更新：每天 08:40

## 📋 涵蓋的資料項目

根據圖片需求，系統已設計支援以下所有資料：

### ✅ 三大法人買賣超
- 外資、投信、自營商買賣超（加權指數、櫃買指數）
- 自營商細分：自營自買、自營避險

### ✅ 融資融券餘額
- 融資餘額（億元）及增減
- 融券餘額（張）及增減
- 借券賣出（張）及增減

### ✅ 期貨未平倉量
- 外資、投信、自營未平倉口數
- 十大交易人未平倉
- 十大特定人未平倉
- 散戶未平倉（計算得出）

### ✅ 選擇權未平倉量
- 外資、投信、自營未平倉口數
- P/C 比（成交量、未平倉）

### ✅ 選擇權履約價分布
- 周選擇權各履約價買權OI、賣權OI
- 月選擇權各履約價買權OI、賣權OI
- 履約價範圍：26800-28400（可配置）

### ✅ 歷史資料
- 期貨交易及未平倉口數（近30日）
- 選擇權交易及未平倉口數（近30日）

## 🚀 使用方式

### 1. 初始化資料庫

```bash
cd backend
source venv/bin/activate
python -c "from database.session import init_db; init_db()"
```

### 2. 手動觸發資料抓取

```python
from database.session import SessionLocal
from crawler.taifex_web_scraper import TAIFEXWebScraper

db = SessionLocal()
scraper = TAIFEXWebScraper()

# 抓取今天的資料
scraper.get_futures_daily_report()
scraper.get_institutional_trading()
scraper.get_twse_margin_trading()
```

### 3. 測試 API

```bash
# 啟動後端
python main.py

# 測試 API（需要先有資料）
curl http://localhost:8000/api/futures/daily?symbol=TX
curl http://localhost:8000/api/futures/institutional?market_type=weighted
```
