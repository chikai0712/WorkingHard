# 資料來源 API 說明

## 資料來源分類

### 1. 融資融券資料 ✅

**資料來源**: 台灣證券交易所 (TWSE)  
**API 端點**: `https://www.twse.com.tw/rwd/zh/marginTrading/MI_MARGN`

**包含資料**:
- ✅ 融資餘額
- ✅ 融券餘額

**狀態**: ✅ 已實現並正常運作

### 2. 三大法人買賣超資料 ⚠️

**資料來源**: 台灣期貨交易所 (TAIFEX)  
**API 端點**: `https://www.taifex.com.tw/cht/3/futDataDown?down_type=2`

**參數**:
- `down_type`: 2（表示三大法人買賣超）
- `queryStartDate`: 開始日期
- `queryEndDate`: 結束日期

**包含資料**:
- 外資買賣超
- 投信買賣超
- 自營商買賣超（含自買、避險細分）
- 三大法人合計

**狀態**: ⚠️ API 端點已確認，但資料解析邏輯未完成

**問題**: TAIFEX API 可能回傳 HTML 重定向頁面，而非直接 CSV 資料

## TAIFEX API 問題

### 可能的問題

1. **HTML 重定向**
   - TAIFEX API 可能回傳 HTML 頁面而非 CSV
   - 頁面包含 JavaScript 重定向

2. **需要 Session/Cookie**
   - 可能需要先訪問網頁建立 Session
   - 或需要特定的 Cookie

3. **編碼問題**
   - 使用 Big5 編碼
   - 需要正確處理編碼轉換

4. **需要網頁爬蟲**
   - 如果 API 不提供直接資料
   - 可能需要使用網頁爬蟲從 HTML 中提取資料

## 解決方案

### 方案 1: 直接使用 API（如果可用）

1. 分析 TAIFEX API 實際回應格式
2. 實現資料解析邏輯
3. 處理編碼和格式轉換

### 方案 2: 使用網頁爬蟲

1. 訪問 TAIFEX 網頁
2. 使用 BeautifulSoup 解析 HTML
3. 提取所需資料
4. 儲存到資料庫

### 方案 3: 使用第三方資料來源

1. 尋找提供這些資料的第三方 API
2. 評估資料可靠性和成本
3. 整合到系統中
